【柳沈】双修搭子 14

发表于 2024-05-08 分类于 Text

柳清歌x 沈清秋（沈垣）
冰妹单箭头沈老师
接之前面第11章，我终于找到让柳沈HE的故事线了，还能顺便开柳沈车~~~
全文接柳清歌和沈清秋打魅妖的结尾。假设魅妖的药非常给力。

柳家爹妈吃完饭就走了，柳清歌还再三叮嘱，此事不可外传，彻底切断跟这园子的联系，驿站和送货人都换一批生人。洛冰河若是知道此地，整个柳家都要陪葬。
饭后，两人在默园的庭院中散步，来到一处开满紫藤的水榭，一水之隔则有一处戏台，每到夜晚便自动亮灯，叮叮当当唱起不重样的傀儡戏。
紫藤从飞檐垂下，若有似无的挡住外界的视线，而透过紫藤又能隐约看到外面的戏台，戏台上正演着才子佳人的木偶戏，木偶的做工极精致，目光能灵活流转，木偶之间两情相悦，眉目动情之处，沈垣心思异动侧头亲了柳清歌眼角的泪痣，让对方微微一颤。
“别在我家放肆。”柳清歌瞪了沈垣一眼，语带威胁的说。
“要是能在这做。。。”沈清秋摇了摇折扇。
“别说了。”柳清歌眉头紧锁面色通红，赶紧捂住沈垣的嘴。
沈垣想到，这人脸皮薄，哪怕是被唱戏的木偶看着，都很害羞，不过越是这样，越是让人忍不住逗他。
戏台上又升起一层小戏台，又演起了皮影戏，居然是戏中戏，戏台上的才子佳人爱恨离别，戏中戏的皮影戏却是大团圆结局，夸张的喜庆画面衬得悲景更悲。
戏演完了，沈垣多少有点后悔，这宅子够隐蔽，可只有傀儡实在太无聊，等自己把这几十套自动傀儡戏看完，也不知道还能干点啥，这里又没有互联网，没法当死宅，还是要出去转转散心，不过要从长计议，设计一个更精密的隐蔽身份，还要用易容术让洛冰河认不出自己，想来依托柳家的势力应该不难，但绝不能给这家人添麻烦，混进松散的天一观是个不错的选择。
回到主卧，小傀儡们已烧好了热水，沈垣脱了衣服走进青玉石砌成的池子，这池子仿佛小型游泳池，四周立着数十根红烛，一侧龙头咕嘟咕嘟吐出着热水，另一侧边缘修成缓坡，可以躺在上面半梦半醒的沐浴，有钱人真会享受，柳大腿放着这种日子不过，非要去外面风餐露宿降妖除魔。
在温暖的热水里，沈垣浑身松弛的半躺在倾斜池壁上，听见水声，转头看到柳清歌也进了池子，温暖的烛光中，水汽氤氲，面容柔和了许多，如墨长发半湿贴着脖颈而下，散在在肩上。
之前抢尸体留下的伤残余了一些，有的已经淡了，有的还是新的，他这五年是怎么过来的，之前听边境的小魔物说，每隔几个月就会有个俊美的仙师来扫一遍幻花宫，试图抢走沈清秋尸体。他一直在战斗，养伤，修炼的循环中吗？每次伤好的差不多，就会继续去战斗，就像推石头的西西弗。
为什么要做这种事情呢？为什么要在那个岛上立衣冠冢呢？
沈垣走过去轻轻抚摸对方背上残存魔气的伤口，注入灵气加快伤口愈合，皮肤下肩胛骨随着呼吸一起一伏。窗外淅淅沥沥下起雨。
“对不起。”无论如何，假死这件事。。。
“抢尸体这件事，无论我们有没有这层关系，事关苍穹山，我都会去的。你不用为此道歉。”柳清歌语气公事公办，的确，十二峰峰主任意一人的尸体落入敌手，他都会去抢回来，似乎没有什么不同。
沈垣转过身，看着直视柳清歌的眼睛：“那天在花月城，我走之前说的话。”
“我听到了。只是不确信，我以为你在安慰我。你设计了自己的死亡，为了不连累我们。”
“不是的，我没有那么好，我只是想逃。。。”沈垣就被亲吻堵住了嘴。
“逃也不是坏事。”在亲吻的间隙，柳清歌轻声说：“至少你回来了。”
在这个陌生的世界，还有人期待着真实的他回来，沈垣吻上他眼角的泪痣，手轻轻拂过对方半湿的长发：“我。。。我。。。”有些话粘在喉咙里，就像是被层层的水雾打湿了，变软了。
“嗯。”柳清歌似乎听到了没说出来的句子，将沈垣拥入怀中，抚摸着温水中这全新的身体。雨越下越大。
“轻一点，这具身体还是第一次。。。”露华芝构建的身体过于新，有成年人的外观，和几乎没有刺激过的皮肉和神经，以至于轻微的触碰都非常敏感。
沈垣向后依在倾斜的池壁，整个身体半浸在在水中，被人缓缓亲吻着，一直到下腹，还没有被蒙眼，难道这一次可以不蒙眼吗？对方是不是在自己假死期间，一直后悔，所以这次。。。就。。。他的下体突然被含住，他抬头想看此刻柳清歌的脸，含着他的那活，向上看着注视着他的样子一定很色，光是想想都硬了，然而对方手一挥，熄了整个浴室里的蜡烛。
雨夜中没有月光，终究又是一片黑暗。
“你。。。你。。。啊。。。”在下面温软吸吮的刺激中，沈垣在水中痉挛，双腿就像抽了筋乱蹬，水花四溅，他双手向下摸到对方浸在水中的长发，手指从水中柔顺的长发中穿过，想抓住，却什么都抓不住。
他很快缴枪，虚脱的躺在水里，初次高潮后，全身都变得极为敏感，即使轻微的水流都能让他颤抖。
那人小心的开拓着他下面的小嘴，那处被轻轻填入手指，慢慢的扩大，还拿了浴后涂身体的玫瑰膏塞进去，让那一处滑腻而湿润。
“可以了，啊。。。”沈垣被弄的有些难耐，但是对方执着的塞进去第四根手指，直到那处能容纳下这样的尺寸，而后圆头慢慢在外面轻轻蹭了蹭。
沈垣张开腿，主动贴了上去，紧接着被压进水里亲吻着，他伸手抱住上面的人，张开双腿缠住对方。
“啊。。。清歌。。。”他叫了一声，伸直脖颈，躺在水中，那东西带着水插进他身体里，饱胀感让他一阵酸麻，那处不听话的一圈圈收缩。
那处渐渐适应之后，对方才开始慢慢插拔，沈垣挺着在水中缓缓起伏着，感受着下面带着水的律动，和脖颈麻麻略微刺痛的亲吻，他伸手抱着对方身体，全力凑上去，脸靠在对方脖颈轻轻蹭着，从喉咙深处发出有节奏的轻柔呻吟声，就像发情的母猫。
雨越下越大，发出震耳欲聋的雷声，远处天幕一道闪光撕开如漆的黑夜。
沈垣只看到眼前人缕缕湿润的发丝粘在锁骨上，他想抬头然而闪光已经消失了，那锁骨太突出了，所以。。。所以。。。柳清歌比以前瘦了。
他在越发强烈的律动中看着黑暗中的残影，轻轻亲吻着对方的锁骨，手顺着对方的腰一路摸上去，瘦了，真的瘦了，本来都很匀称的身体，现在更加精瘦了，肌肉反而又结实了一些，体内流转的灵力也更加充沛，就像把血肉之躯磨成一把锋利的剑，去掉所有不必要的部分。。。因为是凡人，只能靠着一天又一天枯燥的修炼，去对抗天生的血统，屡战屡败，又屡败屡战，为什么要做这些呢？
沈垣坐起来紧紧抱住怀里的人，嘴贴上去亲吻着那反复受伤的身体，那里酥麻着不断收缩迎合着对方，交合越发激烈水花四溅，在轰隆的雷声和忽明忽暗的闪电中，两人在水中紧紧相拥。
有好几次电光中，他似乎能看到对方的脸，但是他眼睛却是模糊的，温热的眼泪抑制不住的流下来。。。只有闪电中模糊的苍白影子，就像暗夜中一闪而过的剑光，或者柳清歌本来就是一把单纯的剑，原本心如琉璃，不染尘埃，只是自己让他多了一些本不应有的情欲。
“清歌，清歌。。。”或许自己就是剑鞘吧，他张开双腿缠住对方，就像剑鞘拥抱着剑。
在高潮的时候，他头垂在对方的肩膀上哭着，这眼泪滴在对方身上顺着满是伤口的脊背流入水中，最深处被射入微凉的液体。黑暗中，对方捧着他的脸，深深吻着，灵力自然的在两人身体流转，手指触碰之前落下层层叠叠的伤口正在迅速愈合。
这一刻，沈垣感觉到真正的从土里爬出来，复活了，以后将作为沈垣（绝世黄瓜）活下去。
片刻之后，他被抱到床榻上，躺在被子里清醒了一些，又觉得太TM丢人了，性生活掉眼泪可不是自己的风格呀，还好当时一片黑，柳清歌大概没看到。
