#摩诃婆罗多# 林居篇总结 札记

发表于 2015-07-05 更新于 2019-02-02 分类于 Text

#ifndef 林居篇总结.md

林居阶段故事脉络：

难受 => （阿周那升级武力）and（坚战云游圣地） => 香醉山重逢 => 回双林附近等待十三年结束。

 

林居经历从下面几个方面概括。

 

### 修行

 

阿周那提升战力和坚战云游圣地，都是在修行，从物理和精神两方面提升。

十三年林居，最大的转变是让五子的心智更加成熟。

坚战一路上听过无数的故事，为了救怖军身处险境。

阿周那在天国选择与兄弟复仇，而不是升上天国。

怖军在蟒蛇面前的自省。

都是逆境中才有的成长，这些光芒夺目的特质就藏在他们心中，但危难之中方才显现。

 

### 婆罗门

 

婆罗门在林居十三年中无微不至的照顾五子，在每个时间节点帮助和指点他们，也用渊博的比惨故事故事安抚他们，滋养他们的心灵。

 

### 盟友黑天和猛光

 

盟友黑天在十三年中，看望五子三次。

第一次是在林居第一年，第二次是在坚战云游途中，第三次在林居即将结束。

在五子整个林居过程中，黑天关心五子，为他们所受的苦难流泪，并拿起武器站在五子这边，支持坚战所做的决定，还和猛光一起帮他们抚养孩子。

 

### 意外

 

黑公主两次被劫，以及n次罗刹侵袭，都表明林居生涯非常危险。

强悍如怖军也在蟒蛇面前陷入危机，被坚战用机智问答救回。

五子林居中最大的危机发生在第十三年。掌管水源的药叉毒死四子。坚战通过机智问答救回四人。

进入香醉山的时候，遇到了自然灾难，黑公主晕倒。

### 生活条件

林居苦难，但是五子还是带了一些军队，仆役和厨师。

第一年林居，比较艰难，似乎没有厨师和仆役，坚战和黑公主一起准备饭。

森林篇：

林居第一年，黑天看望五子，黑天走后，五子的生活水平提高了。

野人篇：

坚战四子和毛密一起游览圣地时，带着厨师。

朝拜圣地篇：

四子前往雪山与阿周那会和的时候，将随从，厨师等人交给妙臂。

战药叉篇：

四子和阿周那汇合之后，再次与妙臂王相见，与厨师和侍卫重逢。

蟒蛇篇：

林居最后两年，在双林和达摩耶伽林居住时，厨师也在身边。

摩根德耶遇合篇：

盗耳环篇：

虽然有厨师，车夫等人跟随，但是五子还是需要打猎，还是要穿着粗衣跋山涉水，被人掠夺，遭遇自然灾害。所以还是很辛苦，只是比缺衣少食要好一点。

另外我还记住了御者帝军（跟帝释军是不是一个人），他收起公主的衣物和首饰带着乳娘和侍女跟上车队。

## 林居篇年表：

//整理的比较粗糙，如有问题请留言。

 

第一年：

    林居生活痛苦而危险，婆罗门和黑天安慰五子。

第二年：

    第十三个月，五子进行未来规划讨论，坚战主和，怖军和黑公主主战。

    毗耶娑到来，指点五子离开双林，给予阿周那升级途径。

    五子离开双林来到伽摩耶加林。

    阿周那见shiva，升到天国，提升战力，获得法宝。

    四子再次讨论未来，怖军黑公主再次主战。

    巨马仙人到来，传授坚战赌博术。

    在毛密仙人陪伴下，四子遍游印度圣地。//在 十一月末十二月初动身出发。

    游览途中，黑天等人看望五子。

    四子来到香醉山，怖军为黑公主在财神花园取莲花。

    坚战黑公主双子被劫。

第五年：

    坚战等人来到白山等待阿周那。

第六年：

    阿周那归来。

第十一年：

    五子讨论未来，离开香醉山。

第十二年：

    五子在毗沙卡逾波树林居住一年。

    怖军遇蛇，被坚战救回。

    五子来到双林，娑罗私婆蒂河边，渡过雨季。

第十三年：

    秋天，五子前往伽摩耶加林，黑天与摩根德耶仙人来看望。

    难敌到双林嘲讽五子，反被健达缚俘虏，被五子救下，释放。难敌试图绝食，被檀那婆说服。

    难敌回城举办毗湿奴祭。

    五子在双林一年零八个月，因鹿梦迁居迦摩耶伽树林。

    黑公主被劫，又被怖军救走。

    因陀罗骗走迦尔纳的金甲和耳环。

    五子再次回到双林，遇到喝水即死的药叉。四子死亡，坚战答对问题救回家人。

    五子准备第十四年的潜伏。

PS：

1. 第二年到第五年之间发生的诸多事情，找不到具体时间。

2. 第十三年五子在双林和伽摩耶加林之间迁移比较混乱，黑天和摩根德耶看望五子的时候，五子在伽摩耶加林，但是难敌嘲讽五子和梦鹿事件里，五子又在双林，他们是什么时候回去？文中并没有提。

也可能文中的顺序，是讲述顺序而不是时间顺序。难敌嘲讽五子和坚战梦鹿都发生在五子在双林的时候。
