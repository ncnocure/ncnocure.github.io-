#7-出生篇（一）##摩诃婆罗多#（豆扇陀国王，你为何拔diao无情不认人）

发表于 2015-02-17 更新于 2019-02-10 分类于 Text

#ifndef 7-出生篇（一）

 

厉声说：

{

    镇群王问及俱卢族的世系。

    护民子讲述婆罗多族起源：

    {

        补卢族的建立者是豆扇陀。//补卢与俱卢是否相等？

        豆扇陀执掌四方土地，国泰民安。

        豆扇陀本人从武力到智商都是人生赢家。//形容国王，形容婆罗门各自都有套路了

        豆扇陀带着军队一路打猎。//其实是对动物的大屠杀

        最终来到一个非常美丽森林。

        森林里有一个森林道院，里面很多仙人。

        豆扇陀走进道院，遇到一位美丽的姑娘。//沙恭达罗上线

        豆扇陀对姑娘一见钟情。

        姑娘自我介绍：我是干婆尊者的女儿。//干婆尊者也是迦叶波的后裔。

        豆扇陀：尊者不ooxx，你怎么可能是他的女儿。

        姑娘自我介绍：

        {

            曾经有位仙人到访时问过我的身世。

            //这里并不是直接讲沙恭达罗的身世，借用尊者回答仙人的话语说出这个故事

            尊者如是回答：

            {

                众友仙人苦修。

                因陀罗又担心他会取代自己。

                他派遣天女美那伽：

                {

                    请你帮我下凡引诱众友仙人，毁掉他的苦行。

                }

                美那伽：

                {

                    他很厉害。我很怕他。

                    百子就是被他所杀。

                    介绍众友仙人有多牛叉：

                    {

                        他出身武士家族，依靠努力成为婆罗门。

                        他为了沐浴造出一条河流。

                        众友送任意行上天事件：

                        {

                            众友背运时，国王任意行曾经赡养众友的妻子。

                            众友仙人出于感激为任意行祭祀，送他上天。

                            天神拒绝任意行，众友用法术将他定在半空中，化作星辰。

                        }

                        他还造出了星斗。

                        他易生嗔怒。

                        他一放神光就会焚毁世界。总之就是各种强力，各种可怕。

                    }

                    我害怕众友，但是既然因陀罗你命令我，我只能去。

                    但是要得到你的帮助。

                    当我嬉戏的时候，请风神脱去我的衣裙。

                    爱神也要成为我的帮手。

                }

                因陀罗应允。

                美那伽诱惑众友：

                {

                    //这一段生香活色。

                    美那伽在仙人面前玩耍。

                    风吹去她素白如月的衣袖。

                    美那伽追着衣袖，玉体尽裸。

                    众友瞬间精虫上脑，与美那伽ooxx。//原文：他向美那伽招呼了一声，全身美丽无暇的女郎便扑向了他。

                }

                仙人和美那伽生下沙恭达罗。

                美那伽将婴儿放在河边，回到天帝宫廷。

                河边有野兽，但是一群大鸟保护了婴儿。//这鸟就是沙恭达罗鸟，沙恭达罗也得名于此

                我/*干婆尊者*/发现了婴儿，成为了她的父亲。

                //印度对父亲定义很有趣，原文：赎身者，救命者，给予饭食者，有资格被成为父亲。

            }//尊者讲述沙恭达罗身世结束

        }//姑娘的自我介绍结束

        豆扇陀爱上了沙恭达罗，向她求婚：

        {

            介绍印度婚姻种类：

            {

                梵式：父亲把男子请回家，把女儿嫁给他。

                天神式：把女儿嫁给一位正在做法事的祭官。

                仙人式：接受新郎一对或两对牛，嫁出女儿。

                生主式：献礼之后，让新郎新娘跟着说一句：“愿双双共同奉行正法。”

                阿修罗式：随意给亲属赠送财物之后娶姑娘。

                健达缚式：男女双方自主结合。

                罗刹式：以暴力抢走姑娘。

                毕舍遮式：强迫姑娘成婚。//目测就是qj

            }

            刹帝利可以用前六钟结婚方式，所以豆扇陀用健达缚式合法迎娶沙恭达罗。//也就是男女自主结合，ooxx

        }

        沙恭达罗开始谈条件：

        {

            如果我生下儿子，就应该成为你的继承人。

        }

        豆扇陀答应了。两人ooxx。

        豆扇陀拔diao离去，许诺会迎娶沙恭达罗。//分分钟打脸

        仙人回家，认可了沙恭达罗与豆扇陀的结合，并告知他们的儿子将会非常伟大。

        沙恭达罗祈求干婆的恩典：

        {

            愿恪守正法的补卢族王业永固。//总觉得被打脸了。。。

        }

        沙恭达罗生了一个儿子，果然很厉害。

        沙恭达罗带着儿子到象城找豆扇陀。//那个时候似乎还没有象城

        豆扇陀果然拔diao无情不认人，完全想不起来了。//另一个版本是受了诅咒才想不起来的

        下面这一对开始了撕逼大战：//太精彩了

        {

            豆扇陀：你这个荡妇，我跟你一点关系都没有。//荡妇是原文。

            沙恭达罗：//这段回骂超级长，畅快淋漓

            {

                你居然这样说我，你到底有没有眼睛。那罗延无所不在，也隐藏在你心里，你在他眼前干坏事，不怕遭报应吗？

                我的娃是你，我已经跟你结婚了。你居然不认他。

                我是天女的女儿。却被母亲抛弃，如今又要被你抛弃。

                可儿子是你亲生的，你不能不认他。

            }

            豆扇陀：

            {

                你是说谎，我不认识你，也不认识你儿子。

                心肠冷酷的荡妇美那伽是你妈，你被她抛弃。

                冷酷无情的众友是你爸，还是个色鬼。

                你是他们两的女儿，谈吐就像娼妓。

                滚吧，假扮苦行女的荡妇。

                你儿子长那么大了，短时间内，他怎么可能那么大？//天神孩子长的快啊。。。

                你生于苟合，非常低贱，我根本不认识你，滚吧。//骂人别骂 爹妈

            }

            沙恭达罗：//沙恭达罗骂人真是充满哲理

            {

                我妈妈是天女，我爸爸是仙人。

                我出身比你高贵多了。

                你轻视自己亲身儿子，就是有违正法。

                我可以离开，但是你必须认你的儿子。

            }

            //故事发展到这里矛盾激化了，大概作者也编不下去了，出现解围精灵

            解围精灵：

            {

                豆扇陀，这娃确实是你的，你要好好养他。

                你娃就叫婆罗多。

            }

            豆扇陀非常开心：

            {

                我就知道这娃是亲儿子，但是如果仅凭沙恭达罗的几句话就认这个娃，

                世人可能会猜疑，认为他不是那么纯洁。//这逻辑，我已经无力吐槽了

            }

            豆扇陀安抚了沙恭达罗：

            {

                我跟你结合，没有人见证，所以很难证明你的清白。

                你十分气恼说了那些难听的话，我会原谅你的。//沙恭达罗原谅你吗？

            }

            豆扇陀为儿子婆罗多进行灌顶仪式。

        }//这场精彩绝伦的撕逼大战以豆扇陀认儿子终结

        豆扇陀之子婆罗多征服天下，被称为转轮王，盖世大帝。//这称号会不会被和谐

        从此婆罗多族崛起。

    }//婆罗多起源，即沙恭达罗和豆扇陀的故事结束

    //7-出生篇，待续未完。

}

#endif

这故事可以概括为“豆扇陀国王，你为何拔diao无情不认人。”

不过豆扇陀的确比罗摩好一点，他预知到男女自愿结合的婚姻会被世人猜疑。

所以他坚持不认儿子，直到外界力量告知真相，以堵住世人的口。

无论那场撕逼大战多么伤感情，至少带来的结果是好的。

但是另一种可能是，他真的不负责任，直到上天派来一个精灵，更有可能是干婆尊者和众友的婆罗门背景导致他不得不认下这个孩子，否则就会触犯婆罗门阶层。（细节：沙恭达罗到象城的时候，身边跟随干婆尊者的徒弟。）

另一方面沙恭达罗在面对丈夫辱骂的时候，非常高贵，她虽然愤怒，但是并不祈求。回骂时条理分明，语言优美，富有哲理，而且紧紧抓住一点：

“你不认我可以，但你必须认自己的儿子。”//我估计沙恭达罗已经对豆扇陀不抱希望了。

豆扇陀只会用“荡妇”“你妈是荡妇，你爸冷酷无情”之类下三路的手段来攻击沙恭达罗。

如果语言可以说明一个人的境界，那么沙恭达罗的境界真的比豆扇陀高一档。

《摩诃婆罗多》沙恭达罗与豆扇陀那段对话，挺值得阅读原文，即使其中的哲理比较难懂，但是沙恭达罗在面临侮辱时候的高贵如此美丽。

7-出生篇里目前看到的故事都挺“知音体”的，下面一段比较长的故事是印度神话里罕见的宫斗戏：

“闺蜜上了我的老公的床，我该怎么办？”
