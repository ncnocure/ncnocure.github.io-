#摩诃婆罗多# 36-蟒蛇篇 札记（ 诺基亚王子被蛇缠住，中了虚弱buff，危在旦夕）

发表于 2015-06-15 更新于 2019-02-04 分类于 Text

#ifndef 36-蟒蛇篇.md

//怖军第一次陷入险境，被坚战救回。。。这相当于孙悟空遇险，唐僧将他救回。

//借蛇和坚战的对话，表述摩诃婆罗多的种姓观念。

 

厉声说：

{

    护民子说：

    {

        五子相聚之后，在森林里安静居住了四年。加上之前的六年/*包括第一年和阿周那学艺的五年*/已经有十年。

        五子在一起讨论之后要做的事情。

        怖军说：这是第十一个年头，只要能骗过难敌，我们就可以复仇，夺回王国。

        '坚战知道了他们的心思。'做出决定，向财神俱比罗道别，带领众人上路。

        五子在毗沙卡逾波树林居住一年。在这里怖军遇到一条蛇，绝望迷乱，内心痛苦。而坚战化身救星把怖军救回来。//第一次啊，坚战救怖军

        他们在树林里渡过第十二年，再到双林。

    }

    好奇宝宝镇群王：

    {

        怖军力大无比，怎会面对蛇惊慌失措。//无敌的诺基亚小王子终于遇到对手了。

    }

    护民子说：

    {

        怖军在森林里游玩，突然被一只巨蛇袭击。

        这蛇能使一切生灵颤抖，犹如死神阎摩。

        怖军被它接触，就神志迷糊，失去力量。

        原来这蛇是从天空坠落的友邻王，蔑视投山仙人被诅咒为蛇。诅咒同时伴随神力，蛇可以抓住任何猎物。

        诅咒的尽头是有人能回答出蛇的问题。

        惯用武力解决问题的怖军第一次陷入弱者的境地，他被蛇缠住，不能动弹，哀叹：我并不生你气，也不抱怨自己。任何凡人怎么可能躲避命运呢？我相信命运是至高的，个人的企图是徒劳的。但是我伤心的不是自己的毁灭，而是我失去王国的兄弟们。得知我死去，他们会失去勇气。'他们恪守正法，由于我渴求王国，他们才遭受磨难。'

        怖军久去未归，坚战感到不详，他将黑公主托付给阿周那，将双生子托付给众婆罗门，去找怖军。

        他见到被蛇缠住的怖军，回答了蛇的问题：

        {

            Q：婆罗门是什么？

            A：达到了梵，无乐亦无忧，就是婆罗门。

            Q：无乐亦无忧的东西，我认为不存在。

            A：谁能表现出世间有不冷不热的东西，所以无乐无忧境界也存在。

            Q：如果在首陀罗身上，见到无怒，仁慈，又当怎样？

            A：当一个人表现出婆罗门的行为时，他就是婆罗门。

            Q：你根据行为判断婆罗门，那么出生之前无行为，又该怎样判断？

            A：种姓都是混杂的，人出生难以断定。但是当他表现出婆罗门的行为时，他就是婆罗门。

        }

        蛇满意坚战的回答，放过怖军。

        坚战向蛇问及无上之道。

        {

            蛇：不杀生，说可爱的话。

            坚战：不杀生和说可爱的话，孰轻孰重。

            蛇：一切依据效果。

            坚战：灵魂升入天国，必然的业报。

            蛇：三种道路，为人，为动物，升入天国。

            坚战：灵魂与五感怎样安然相处。

            /*蛇这段回答位于蟒蛇篇，178：18-27，讲述灵魂和感官的关系，我暂时无法理解。*/

        }

        友邻王解除的诅咒，返回天上。坚战将怖军带回去，责备他以后不可这样。

    }

}

 

#endif

 

##本章节主要内容：

1. 怖军加入蛇族全家桶。

2. 坚战回答蛇的问题，救回怖军。

 

##怖军文中第一次内省

怖军惯于用武力解决一切，我们也乐于见他打砸一切。

这次他第一次陷入险境，即使是这样桀骜不驯，用武力解决一切的英雄，也有印度独有的世界观。

认定命运不可抗，但在死前他想起的是其他的兄弟。想起他们会因为失去怖军而丧失勇气，想起自己给兄弟们带来的不幸。

原著里每一个角色都这么有血有肉，他们曾经活在这个世界上，如今又活在这个故事里。

而拯救怖军的，却是武力值很弱的坚战。

 

##友邻王的故事

友邻王的世系来自人类始祖摩奴：

摩奴=>伊罗=>补卢罗婆娑=>长寿=>友邻=>迅行。

有段时间因陀罗被赶下宝座，友邻王代替因陀成为国王，但是他倒行逆施，触怒投山仙人，被赶下天庭当蛇。
