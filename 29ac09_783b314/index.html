#摩诃婆罗多# 42-黑公主遇劫篇 札记 （永远都有比你惨的故事）

发表于 2015-07-04 更新于 2019-02-02 分类于 Text

#ifndef 42-黑公主遇劫篇

//黑公主被劫，引出罗摩之妻被劫插话

//萨维德丽拯救身边人插话

 

厉声：

{

    护民子：

    {

        信度国王增武之子胜车劫走黑公主。

        五子夺回黑公主。打斗时黑公主给胜车介绍自己的丈夫：

        {

            坚战遵循正法。

            怖军有仇必报。

            阿周那光辉似火。

            //以上还比较正常。

            '人们说他的形体漂亮，在大地上无与伦比，所有的般度族兄弟都保护他。他们待他胜过自己的生命，这就是我的丈夫无种。'

            偕天：心灵手巧，精通武艺，聪明睿智，最年轻，最可爱，善于判断，善辩，不屈不挠。

            总之般度之子会消灭你们的。

        }

        五子大杀特杀，无种还亲手干掉一只大象。//脑补精灵王子干巨兽那幕

        眼看无力回天，胜车释放黑公主逃掉，而后又被五子抓住。

        阿周那劝怖军看在杜纱罗和甘陀利的份上放过胜车。//杜纱罗是甘陀利和持国的女儿，是胜车的妻子

        胜车宣称自己是奴隶，而被释放。过后他修行大苦行，赢得shiva欢心，望能杀死五子，shiva十动然拒。

 

        坚战看到妻子被劫走，觉得生活不幸。

        摩根德耶，在妻子被劫的比惨大会上，你比不过罗摩。仙人讲述罗摩的妻子被罗刹抢走的故事。

        {

            罗摩出身于甘蔗族。悉多出身于毗提诃。

            罗波那祖父为生主。生主自体繁殖了补罗私底耶。//这里的生主为'自在天，一切世界的主人，具有大苦行的创造者'。可能是大梵天。

            补罗私底耶生吠室罗伐拿离开。吠室罗伐拿离开父亲来到祖父身边，成为财神俱比罗。

            补罗私底耶又生四子一女，其中一子名罗波那，长有十个头，又名十首王。

            罗波那修行大苦行，让人以外的种族都不能战胜自己。

            获得恩赐之后，罗波那打败财神，将他赶出楞伽城。//财神也因此长期定居香醉山，并遇到坚战等人

            十首王罗波那成为罗刹和毕舍遮的王，让众生哭泣。//罗波那就是让人哭泣的意思

            梵天让毗湿奴下凡杀死罗波那，又让因陀罗为首的众天神与母猴和母熊生下儿子，作为毗湿奴的助手。//为罗摩故事也加了一个宏大背景，跟摩诃婆罗多好像

            梵天还命一位罗刹女下凡挑拨是非。

            十车王之子罗摩和罗什曼已经长大。罗摩本为继承者，然而罗什曼的母亲受罗刹女挑拨，向十车王请求恩惠，让罗什曼继位，让罗摩去森林。

            罗摩为了让父亲信守承诺，自行去了森林，罗什曼和悉多跟随他一起去了森林，十车王死去，罗摩也未曾归来。

            十首王与罗摩结仇，劫走悉多。

            罗摩在猴王妙项和哈奴曼等猴军以及财神帮助下夺回悉多。

            罗摩怀疑悉多已失身，“不管你的行为端正不端正，我不能再享用你，犹如被狗舔过的祭品。”

            风，火，水，梵天和十车王都来证明悉多的贞洁，并证明罗波那曾受诅咒，qj不从的女子，身体就会裂成百块。

            罗摩终于接受悉多，并回国统治自己的国家。

        }//罗摩妻子被劫故事结束

        摩根德耶以这个故事来安慰妻子同样被劫的坚战，告诉他，你有出色的帮手，不需要发愁。

        坚战认为自己妻子大福大德，忠于丈夫。

        摩根德耶说萨维德丽也是大福大德忠于丈夫：

        {

            国王无子，他修苦行求子，得到一个女儿萨维德丽。

            萨维德丽长大之后，拥有世间一切美和美德。

            萨维德丽找到一个可心的丈夫，他是沙鲁瓦国王的儿子萨蒂梵。沙鲁瓦国王双目失明被邻国仇敌夺走王国，正定居在林中。

            萨蒂梵则是集合了聪明，智慧，勇敢，宽容，美貌，柔顺，温和，勇敢，诚实等等一切好词。然而唯一缺点是从今天算起寿命还有一年。

            即使这样萨维德丽还是嫁给了萨蒂梵。

            随着一年期限到来，萨维德丽不断修行苦行，希望能拯救丈夫。

            当阎摩来接萨蒂梵的时候，萨维德丽跟着阎摩直到求得恩惠，让公公的双目复明，恢复国家，也让自己的父亲有一百个儿子，让自己有一百个儿子，最终复活自己的丈夫。

        }//萨维德丽的故事结束，这个故事挺感人的，尤其是萨维德丽为了爱人修行大苦行，公公知道儿媳在做什么，感觉痛心，又不能阻止她

 

        摩根德耶：萨维德丽将所有人救出困境，同样黑公主吉祥有福，也会像萨维德丽一样挽救你们所有人。

        在摩根德耶的开解下，坚战解除忧愁住在迦摩耶伽林中。

    }

}

#endif

 

##主要内容

1. 黑公主被劫

2. 仙人摩根德耶用“罗摩妻子被劫”/*世上有比你更惨的*/

    和“萨维德丽拯救身边所有人”/*黑公主可以拯救你们*/

    的插话安慰坚战。
