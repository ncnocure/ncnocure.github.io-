#22-诛妖连篇# #摩诃婆罗多# 札记（有没有婆罗门大大出本，决定了历史地位。。。）

#ifndef 22-诛妖连篇.md

//小葵花跟沙恭尼好像，两人都洞悉人性弱点

//妖连其实是摩揭陀的贤王

 

厉声：

{

    护民子说：

    {

        黑天给出杀妖连计划：

        {

            汉沙，狄婆伽，刚沙已死。杀妖连的时机已到。在战争中，天神和阿修罗都不能战胜妖连，所以应非战争场合以单挑方式杀死妖连。//因为妖连军队强大，直接开战无法战胜妖连。只能通过与妖连决斗，来杀死他。

            我们需要在无人的地方与妖连相遇，刹帝利绝不会拒绝挑战，他会和我们中一个交战。'妖连蔑视世界，一旦受到激将，就会和怖军搏斗。'怖军会杀死他。//赌骰篇中，沙恭尼有相似的话：坚战喜欢掷骰子，又不精于此道，只要有人邀请他，他就难以拒绝。

            坚战，若你信任我，请把阿周那和怖军交给我。

        }

        坚战被黑天劝服，又看到阿周那和怖军充满喜悦，答应黑天诛杀妖连的请求。

        诛杀妖连始末：

        {

            黑天，阿周那，怖军扮作刚完成学业的婆罗门。来到摩揭陀国京城。

            概述摩揭陀京城

            {

                '摩揭陀国京城又大又美，牲畜兴旺，水源充足，房舍精美，无病无灾'。//妖连把国家治理的很好。

                乔达摩仙人效忠于摩揭陀王朝，几大蛇王居住于此。

            }

            三人以婆罗门身份来到妖连王住所。妖连王曾立下誓言：“'只要听到学习结业的婆罗门到来，哪怕是在半夜，他也要起身去迎接。'”//妖连非常尊重婆罗门，尊重知识阶层。

            妖连向三人行礼，又从细节发现三人假扮婆罗门，妖连质问他们：

            {

                1. 学习结业的婆罗门不带鲜花，不抹香膏，而你们带了花。

                2. 你们手臂上有挽弓的印痕。有刹帝利的豪气却身着婆罗门的衣服。

                3. 不经大门就闯入我们宫殿。

                4. 向你们行礼，你们却不接受。

            }

            黑天回答：

            {

                1. 刹帝利持戒也会有好运。//承认自己是刹帝利

                2. 带花的人肯定吉祥如意，所以我们带花。

                3. 善人从不由大门进入敌人家，只有去朋友家才由门进。所以不由城门进入你家。

                4. 到了敌人住所，不能接受敌人的敬意。

            }

            妖连：我跟你们什么仇什么怨？我是个好人，杀我要遭报应的。

            黑天：你把世间刹帝利都抓起来献祭给shiva，这就是你罪过。我黑天和两个般度之子要代表正法惩罚你。

            妖连：刹帝利的天职就是征服别人。我征服国王们，把他们抓起来祭神有什么错？'今天无论是布阵的军队对军队，还是一个对一个，或者我一个对你们两个，三个，同时或者分别，我都奉陪。'//好帅。。。

            说完妖连王给儿子偕天灌顶。//先交代后事

            战斗即将开始时，妖连王想起了汉沙和狄婆伽。//求婆罗门出本，生死之战前想起好友，到底想到什么？

            黑天没有亲自动手，妖连挑了怖军。'“怖军啊！我要跟你打，因为败在强者手中也是好的。”'

            两人打了十三天，到了第十四天晚上妖连王累了。

            '怖军在黑天指导下杀死了妖连王。怖军用双手举着他，在空中打了一百个转，然后吼叫着扔下，摔断他的脊背。

            粉身碎骨的妖连的惨叫声和般度之子的吼叫声混在一起，令一切众生恐怖异常，使所有摩揭陀人胆战心惊，使孕妇们流产。'//画风变作水浒。

            三人把妖连王丢到王宫门口，而后离去。黑天架起妖连的车子与般度之子一起解救被囚的亲友。

            被囚禁的国王们感谢黑天，送上珠宝和许诺。黑天将这些人拉拢到坚战旗下：'“坚战想要举行王祭，这位遵行正法的国王想要取得帝位，你们在举行王祭的事上帮助他吧！”'

            而后黑天为妖连之子偕天举办登基仪式，让他继承摩揭陀国。

        }//诛杀妖连事件结束

        三人带着被妖连囚禁的国王们回到天帝城，向坚战报平安。被解救的国王告别坚战回到自己的国家。

        诛杀妖连救出国王让五子威望大增。

    }

}

#endif

#本文主要内容：

1. 黑天拿出杀妖连的计划，最终说服坚战

2. 诛杀妖连

##杀妖连的计划

上一章中黑天事实上已经说服坚战，阿周那和怖军to do，但是无法解决how to do。

这个章节黑天分析了妖连的弱点：

    1. 汉沙和狄婆伽，以及刚沙已死，妖连势单力孤。

    2. 妖连骄傲，被挑衅必然应战。如果汉沙和狄婆伽或刚沙在，可能会劝阻头脑发热的妖连。但是这三人已经不在。

 

制定杀死妖连的计划：

    1. 无人的地方与妖连相遇。//避开妖连强大的军队

    2. 与他一对一的决斗。//借助妖连易被挑衅的弱点

    3. 妖连骄傲，会挑选怖军为对手。怖军会杀死他。

对敌人深刻洞察，才能制定这样计划，不久之后同样洞悉人性的沙恭尼也如此算计坚战：'连天神也不能以武力战胜五子，不过，我知道有个办法可以战胜坚战本人。'

##杀死妖连具体实施

而后章节一切也按照黑天的计划进行。

    1. 三人扮作婆罗门，潜入妖连王宫殿，与妖连见面。

    2. 挑战妖连做一对一决斗。怖军杀死妖连。

##杀妖连的影响

 

杀死妖连之后，黑天释放了妖连囚禁的国王，得到极大威望，也为五子王祭做了铺垫。而且他将被囚禁的国王带到天帝城，恩威并施为坚战攒下一大票政治盟友。王祭之时，被黑天拯救国王也纷纷出场，成为坚战盟军。

正如黑天所许诺，杀妖连让坚战横刀立威有了王祭资格。

最后黑天还为妖连之子偕天举办的登基仪式，与摩揭陀下一代政权交好。//PS：这要多高的魅力值才能让偕天对杀父仇人俯首称臣。

##杀妖连和赌骰篇

看过赌骰篇回来记上一笔。黑天制定杀妖连的计划，跟沙恭尼欺骗坚战的计划何其相似。

1. 洞悉对手缺陷。

    妖连-傲慢。

    坚战-好赌。

2. 针对弱点，发出无法拒绝的挑战。

    激怒和挑战妖连。

    大会上邀请坚战赌博。

妖连为一世之雄，坚战亦是举行王祭的正法王，他们的致命弱点被抓之后，他们的霸业如蚁穴般坍塌。

无法与一个团体正面对抗，就针对团体首领的个人弱点，予以攻击，这个手段本身到底算不算符合正法？

 

##妖连其人

 

本章节中对摩揭陀京城的描述，侧面展现了妖连其人：

'摩揭陀国京城又大又美，牲畜兴旺，水源充足，房舍精美，无病无灾。'

'城里住着四大种姓的人，心满意足，身体肥壮，一片节日繁荣景象。'

'只要听到学习结业的婆罗门到来，哪怕是在半夜，他也要起身去迎接。'

治理国家出色，尊重知识阶级，妖连对内贤明，让四大种姓和谐共处。

上篇商议篇中，黑天口中的妖连是个对外强势的国王。//黑天和妖连是仇敌关系。黑天杀死妖连的妹夫刚沙，妖连因此追杀黑天所在部族。

黑天口中妖连的黑点：

'妖连，而他生性愚昧，没有教养，从不满意。 他依靠武力掠夺那些行过灌顶礼的国王，我们没有见到哪儿有人不向他纳贡。'

但是，下一个章节五子就要征四方，令其他国家纳贡，算不算武力掠夺？

【脑洞】如果征四方算'武力掠夺'的话，五子与妖连同样是对内贤明，对外强势。

{

    那么他们唯一区别就是有没有婆罗门大大出本？//所以婆罗门才这么受尊重？

    若妖连方有婆罗门出本。那五子就是耳根子软，被黑天说动杀死无仇无怨的贤王妖连。而黑天不敢与妖连王战场对决，只敢到无人之处，暗算妖连王。

    如果真有婆罗门为妖连一系出本，求at我！

    PS：五子没有监禁其他国王作为shiva祭品，他们使用了王祭这种较为文明的掠夺方式，征四方篇里再细述区别吧（如果有）。

}//脑洞 end

 

无论妖连本人是否贤明，只要跟黑天有冲突，且坚战要举行王祭，肯定要被杀。

妖连的黑点只是让这个杀戮看起来正当一些。
