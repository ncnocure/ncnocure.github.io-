#33-朝拜圣地篇#（1） 札记 #摩诃婆罗多#（足疗按摩哪家强？）

发表于 2015-05-31 更新于 2019-02-04 分类于 Text

#ifndef 33-朝拜圣地篇.md

 

 

//本章节的内容相当于印度圣地旅行指南，掺杂了各个圣地得名的小故事。

//因为插话太多，太长，跟剧情基本没关系，所以先把刨去插画的部分简述一下。下一篇再总结里面的插话。

//亮点大概是，足疗按摩哪家强？以及怖军给黑公主摘莲花。

厉声说：

{

    护民子说：

    {

        大仙以推荐十全大补丸的态度推荐印度各处圣地，如果能对照地图估计是印度宗教游攻略。

        印度圣地有各种功能，例如美容，治病保健，净罪，积攒功德（以一千头牛为单位），得钱，得成功，不为死亡忧伤，等价一千次/一万次的火祭/王祭/马祭/莲花祭/通宵祭，到达太阳世界/风神世界/毗湿奴世界/天国世界/苏摩世界等等。

        在大仙的忽悠下，毛密仙人到来，带来阿周那的传话：“不要担心，我正在努力升级，你们也到圣地旅游，刷刷成就点吧。”

        四子和毛密一起走上了云游印度的旅途，期间掺杂了大量插话小故事。

        中途他们遇到持斧罗摩。

        路程中黑天和大力罗摩带着军队来看望他们，看着他们一路苦修，留下伤心的泪水。

        //木有任何预兆，摩诃婆罗多作者只要写到大力和黑天，就忍不住进入花痴模式。

        '大力罗摩白净似牛奶、茉莉花、月亮和莲藕，带着野花编成的花环，对眼似莲花的黑天'哀叹坚战五子的命运。

        萨谛奇：我们带着军队现在就出发，杀掉难敌等人，将王国还给坚战。

        黑天：你说的没错，只是不凭自己双臂征服的土地，般度五子不会要的。

        坚战：黑天得我心意，请诸位英雄先回去，十三年后誓言达成，我们再相聚取回国家。

        他们互相道别。坚战一行人继续周游圣地。

        为了见到阿周那，坚战一行人在香醉山中遇到风暴大雨。黑公主累倒。坚战表达对黑公主的歉意：

        '由于我这罪人的过错，她没有得到任何幸福。'

        四子亲手照顾晕倒的黑公主。

        '般度之子们用清凉的手一再抚摸黑公主，将可怜的苦行女黑公主放在铺开的鹿皮上，安抚她，让她恢复知觉。孪生兄弟无种和偕天用结满老茧的双手轻轻按摩她的有吉祥标志的、脚底通红的双足。'//足底按摩那家强？

        黑公主醒来，但是前路还有雪山，黑公主没法登上雪山。

        怖军想起自己的儿子瓶首。

        瓶首可以在天空中行走。于是怖军叫来自己的儿子。

        瓶首带着一群罗刹背着四子和黑公主以及毛密等仙人来到美丽的香醉山。

        此地是那罗和那罗延修行的地方。

        一天风吹来了美丽的莲花，黑公主喜欢这莲花，请求怖军多去采摘一些。

        怖军知道黑公主的心意，一心想讨好心爱的人，向着吹来莲花的方向走去，在芭蕉林中遇到沉睡的猴王哈奴曼。

        哈奴曼劝说怖军回去。//大约是调戏自己的弟弟。

        他让怖军挪开自己的尾巴。

        怖军挪不开，他知道眼前的猴子非比寻常。

        哈奴曼表明自己的身份，讲述自己帮助罗摩夺回悉多的故事。

        哈奴曼向罗摩求得恩赐：“'只要罗摩的故事在世上流传，我也就会活着。'”//猴王现在还活着。

        哈奴曼告知怖军，眼前的道路只有天神可以行走。

        怖军得知眼前是哈奴曼非常开心，他想看看哈奴曼跃过大海的样子。

        哈奴曼：

        {

            时代不同，现在是毁灭时代，我无法显现当时的形象。

        }

        哈奴曼讲述四个时代特点：

        {

            圆满时代，正法不会衰落，人民不会毁灭。'一切众生的灵魂都是白色的那罗延。'

            三分时代，正法减少一足，人们守信守法。那罗延变成红色。

            二分时代，正法减少两足，众生从事苦行和布施，也迷恋激情。那罗延变成黄色。

            毁灭时代，正法衰落只剩下一足，人们争斗毁灭。那罗延变成黑色。

        }

        在怖军的要求下，哈奴曼最终变成显现了自己昔日的形象，巨大而占据四面八方。

        怖军：您如此强大，为何没有亲手杀死罗波那。

        哈奴曼：为了成就罗摩的美名，我放过了罗波那。

        哈奴曼指出芬芳林财神花园的路，让怖军向财神献祭再摘花，向自己同为风神之子的弟弟道别。

        怖军'把黑公主的话当作路上的干粮，加快速度前进'，来到财神花园，并没有照哈奴曼的吩咐做，而是打败看守花园的罗刹，采摘莲花。//真。熊孩子

        面对自己重伤的手下，财神笑道：“'就让怖军随意采摘莲花吧！我知道他这样做是为了黑公主。'”

        怖军采花的时候，暴风大作，坚战带着其余的人在瓶首的帮助下来到莲花池，看到怖军。

        '法王坚战见到他后，一再拥抱他，用亲切的声音说道：“贡蒂之子啊！你这是做什么？嗨！祝你幸运！你这样鲁莽，连天神们也不会喜欢。如果你想让我高兴，以后不要做这样的事了。”'//我弟弟就这么吊，我引以为傲啊，以及弟弟，你还是不要这么吊，免得出事。

        般度之子们接过莲花，在池子里玩耍，他们在财神花园里住了很久。

    }

}


#endif


##主要内容

 

1. 大仙人安利印度各地圣地。

2. 毛密见坚战，将阿周那的话带到。

3. 四子和毛密云游各大圣地，讲述每处圣地的故事。

4. 坚战四子在香醉山附近遇险。怖军儿子瓶首率领众罗刹救四子。

5. 怖军为了给黑公主采莲花，长途跋涉，路上遇到猴王哈尔曼。

6. 怖军采到莲花，坚战等人找到怖军，一行人在财神花园里停留。


##四子和黑公主


坚战不太表达感情，但是黑公主晕倒之后，一贯冷静沉稳的他，这次也终于向所爱的女人道歉了：

'由于我这罪人的过错，她没有得到任何幸福。'

'般度之子们用清凉的手一再抚摸黑公主，将可怜的苦行女黑公主放在铺开的鹿皮上，安抚她，让她恢复知觉。'

'孪生兄弟无种和偕天用结满老茧的双手轻轻按摩她的有吉祥标志的、脚底通红的双足。'

旅途虽然辛苦，但是这段好暖。不过问题来了：

足疗按摩那家强？


最亮的当属肉山大魔王怖军，为了老婆摘莲花，饿了有老婆的话当干粮，路上还遇到风神哥哥哈奴曼。


坚战找到他的时候，还亲切数落了他一顿，语言吐槽向，语气却是暖暖，一副我弟弟就这么吊，我引以为傲啊，以及弟弟，你还是不要这么吊，免得出事。
