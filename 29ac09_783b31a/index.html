#摩诃婆罗多# 43-盗耳环篇 札记 （迦尔纳也是个轴人。。。以及因陀罗也是个老狐狸。 ）

发表于 2015-07-04 分类于 Text

#ifndef 43-盗耳环篇

//迦尔纳也是个轴人。。。以及因陀罗也是个老狐狸。

 

厉声说：

{

    镇群王：

    {

        毛密奉因陀罗之命见坚战：

        我将消除你的恐惧。

        坚战的恐惧究竟是什么？

    }

 

    护民子：

    {

        坚战的恐惧就是迦尔纳。

        五子林居的第十三年，太阳神来到迦尔纳身边，请他不要将自己的耳环和铠甲施舍给婆罗门。

        因为因陀罗将扮作婆罗门从迦尔纳手中乞讨耳环。

        然而迦尔纳也是个轴人，他发下誓愿：能将一切施舍给婆罗门。

        即使因陀罗扮作婆罗门来乞讨，迦尔纳也会施舍耳环和铠甲。

        迦尔纳视荣誉为一切，宁愿光荣死去赢得尊重。

        太阳神见儿子如此之轴，就让他用耳环换取百发百中的标枪‘力宝’。

    }

 

    镇群王：

    {

        迦尔纳铠甲和耳环从哪来？

    }

    护民子陈述了贡蒂服侍仙人，获得咒语，怀上太阳之子的过程。

    {

        迦尔纳的铠甲和耳环来自贡蒂的祈求。

        太阳神给予自己的儿子甘露制成的耳环和天神铠甲，让他刀枪不入。

    }

 

    护民子：

    {

        每当中午，迦尔纳站在水中，双手合十，礼赞光辉的太阳。因陀罗出现向他祈求耳环和铠甲。

        迦尔纳猜出他是因陀罗，以标枪为交换条件。

        因陀罗：我把标枪给你，它会杀死一个强大的，吼叫的，发光的敌人，然后回到我手中。

        因陀罗：你将在战场上杀死一个强大的敌人，那个人受到灵魂伟大者的保护。

        因陀罗：如果你手上有别的武器，心不在焉掷出这支标枪，那么它会落到你自己身上。//flag！

        迦尔纳：最大危险的时候，我才会掷出。//flag！

        迦尔纳接受了标枪，自己用到微笑着割下耳环和铠甲，血淋淋的交给因陀罗。

    }

}

 

 

#endif

 

##主要内容

1. 因陀罗从迦尔纳手中骗得耳环和铠甲。

2. 贡蒂获得太阳之子的详细插话。

 

##吐槽

迦尔纳也是个轴人啊。

因陀罗也是个老狐狸，能够玩弄语言。
