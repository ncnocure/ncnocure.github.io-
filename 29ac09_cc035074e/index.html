【柳沈】双修搭子 16

发表于 2024-05-10 分类于 Text

柳清歌x 沈清秋（沈垣）
冰妹单箭头沈老师
接之前面第11章，我终于找到让柳沈HE的故事线了，还能顺便开柳沈车~~~
全文接柳清歌和沈清秋打魅妖的结尾。假设魅妖的药非常给力。

次日，沈垣倚在水榭美人靠上摇着折扇发呆，而柳清歌则照例在院子里练剑，落花中舞动的乘鸾刺向心魔这个假想敌。
到了傍晚，水榭外的戏台又开始叮叮当当的唱戏，这次是西洋戏，王子父亲死的蹊跷，于是王子找来戏班子在嫌疑人叔父面前演出相同的弑君戏码。
在大戏台上上演着王子复仇，血流成河。同时，大戏台上嵌套的小戏台上则是一出喜剧，弑君的凶手被砍头，而被弑的君主则阴差阳错般的活下来跟爱人儿子一起相拥而泣。
沈垣想到一个计策，能不能实现还待定。这天晚上，他将红绳一头系在自己腕部，另一头系在柳清歌手腕上。
“如果我拉绳子，你就用灵力注入百会穴，强行把我弄醒。”
柳清歌点了点头，抱着剑在他身旁打坐。
沈垣枕着他的腿，躺在床上缓缓睡去，虽然是凶险至极的计策，但身边这个人让他安心。
又进了梦境，想来自己只要活着，大概夜夜被洛冰河无意识中拉入梦境吧，自己种下的因果终究绕不过去的。
而这次不是进入沈清秋躯壳中，而是身处炼狱般的景象中，沈垣伸手看了看自己的袖子，果然在洛冰河梦境中，自己还是沈清秋的模样。
血肉撕裂的声音。
还有喑哑的惨呼。
沈垣按紧太阳穴,眼前景象逐渐清晰起来。
一片血海。
尸堆成山。
洛冰河站在这彷如炼狱般的场景中,木然而立。他身穿玄衣,染不上红色，可半边脸颊都贱上了殷红的血色，机械又冷酷地手起剑落，居然在残杀自己的梦境造物，跟自己拿一把钢刀往他脑浆里搅基本没什么区别。看来柳清歌言语中小疯子，所言非虚。洛冰河也真的快撑不下去了，如果作为世界能量源的他崩溃失控，那么仙魔世界会怎么样呢？也如眼前的炼狱一般吧。
“梦魔前辈。”趁着意识模糊的洛冰河还没看到自己，沈垣默默呼唤出这个关键帮手。
一个黑影裹挟着他，将他带到一处竹林，这是洛冰河梦中的清净峰，也是他梦境中唯一的净土，现在他沉湎杀戮自虐，神识目前不会注意到此地，暂时安全。
“你还敢回来？你已经把他害得够惨了。”梦魔皱着眉头看着梦境中沈清秋形态的沈垣。
“我需要您帮忙治疗洛冰河，他的神智撑不了多久了。”
“我为什么要帮你？”
“洛冰河死了，你很难找到第二个出色宿主，也会随之魂飞魄散吧。与其等死，不如最后一搏。”
梦魔深吸了一口气：“你想干什么？”
“我要设计一个梦中梦，治疗洛冰河。”
“在洛冰河的梦里，所有一切都是被他控制，你觉得可以吗？”
“所以要进我的主场，我在他的梦中做梦，然后引他进入我的梦境。”
就像王子复仇记中的戏中戏，沈垣顺手在地上画出一个大圈：“第一步，我进入他的梦境。”
而后他又在地上画出一个小圈：“第二步，我在他的梦境，造出第二层梦境。”
而后，他在小圈边缘画出向内的小箭头：“第三步，引他入我的梦境。”
最后，沈垣在小圈子里画出一颗心：“第四步，在梦境中给他一个幸福的童年，来治愈伤害。梦魔前辈，我需要你在第二步掩护我，并且教我造梦，在第三步时将洛冰河暂时清掉记忆带入我的梦中。”
“小子，梦中梦凶险异常，稍有不慎你就会堕入梦境深处，再也醒不过来。而且，梦中一年，现实一日，再嵌套一层梦境，时间流速会更快，梦中梦十年，现实中可能只过片刻。你要做好困在梦境中十几年的准备。”
“十几年也好，幸福的人用童年治愈一生，不幸的人用一生治愈童年。我改变不了过去，至少可以给他足够多温暖的童年回忆。”
梦魔叹了口气：“那就开始吧，趁着洛冰河现在神识不清。反正就算失败了，状况也不会更差了。”
“等等，前辈，我想请让你多带一个人入梦中梦。”
“易如反掌。”梦魔一笑拂袖。
“他可不太好叫，你要告诉他：梨花烤肉岛岛主请他来。他才会乖乖入梦。”沈垣轻摇折扇笑着说。
