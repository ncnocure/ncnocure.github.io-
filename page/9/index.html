你的哥哥将会为他的怀疑，对我的抛弃付出应有的代价，我得到那罗延天与拉克什米女神的眷顾，在我与我儿子的统治下，我会让它成为世间最强盛的国家，我悉多的名声会在百国传诵，几千几百年后，人们在歌颂我的时候，都会考量着，质问着你兄长的良心，那就是，一个丈夫应该如何对自己的妻子。

上善若水:


艺人拍手笑道：“那么我来讲一讲罗摩与悉多的故事。”

 

艳光不自然的挑了挑眉。

 

“悉多公主得神佑，一朝为女初长成，眉目似莲揽瑶芳，落花随风伴圣君。”艺人打着拍子，结束念白后开始讲另一个故事，“悉多女神伴随圣君身边，苦行十余载，每一个婆罗门仙人都尊敬这位伟大的女子，然而，在册立王储的时候，有一个大臣，提出了自己的疑虑。”

 

“在那天，鲜花与黄金笼罩的国度染上了灰边。”

 

“大臣说道：‘圣君啊，你是知道的，罗波那美容仪而芝兰玉树，你如何能够证明，王妃在罗刹之地那么久的时日，并没有对罗波那动情？’”

 

“圣君笃信自己妻子的清白，但却不得不开口询问，悉多王妃一再辩解，但罗摩迫于朝臣的压力，要求悉多公主投火自明。”

 

“悉多公主性烈如火，自然受不了这种侮辱，人们啊，请记得，她与圣君罗摩共同苦行十余年，将荣华富贵全抛下，面对罗波那的甜言蜜语，她心如磐石，风吹雨打屹然不动，这是何等的刚烈女子。”

 

“她对着拉克什米女神控诉，‘我为了我丈夫付出了一切？这是我应得的？’

 

女神现身，用无上荣光庇护着阿逾陀的公主，她说道：‘我的孩子，做你应该做的事情去吧，无论何时，我与那罗延天都会站在你的身边。’”

 

“阿逾陀的国王发兵要为自己的女儿讨回公道，而罗摩的兄弟跪在自己嫂子的面前，祈求她的宽恕。”

 

“罗什曼那从不抬头看他的嫂子，低头说道：‘我尊敬的嫂子，我的兄长言出必行，他只此一生，只有您一个妻子，您为什么不能看在这些情分上，原谅他？回到他的身边？”

 

“悉多公主却拒绝了他的请求，‘你请回去吧，这场战争仍然会继续，因为这场战争，已经不仅仅是因为我与你兄长的婚变。

 

你兄长作为国王，一言一行为万民表率，然而他如此对我，日后，百姓就会如此对他们的妻子。

 

你的哥哥将会为他的怀疑，对我的抛弃付出应有的代价，我得到那罗延天与拉克什米女神的眷顾，在我与我儿子的统治下，我会让它成为世间最强盛的国家，我悉多的名声会在百国传诵，几千几百年后，人们在歌颂我的时候，都会考量着，质问着你兄长的良心，那就是，一个丈夫应该如何对自己的妻子。’”

 

艳光放下酒杯，悄声问她的丈夫：“我为何不记得有这桥段？”

 

在她印象里，真正的故事是悉多把罗摩打了一顿，夫妻两个兴高采烈的回家去了。

 

“今人之生平爱恨，后人自会用自己理解的方式演绎出来，用自己的标准，寻找每一个问题的答案，然而，并不是每一个问题，一定都有答案。”奎师那笑。

 

沧海桑田，人们永远喜爱的戏剧般圆满结局，因为生活本就艰辛。

 

所以婆罗门仙人在自己的神话里，给了罗摩与悉多一个圆满。

 

“圣君罗摩与悉多女王将国家合并，共同统治着潘地亚，当人们提起罗摩的时候，一定会说，他是一个了不起女人的夫。”

 

艺人鞠躬，罗摩的故事谢幕。

#毗罗吒篇总结

##主要内容

1. 五子换装潜入毗罗吒王城。

2. 黑公主被空竹骚扰

=>怖军解决空竹

=>空竹死

=>俱卢和三穴联手攻打毗罗吒王

=>五子与俱卢三穴对阵

3. 五子恢复身份。

4. 激昂与毗罗吒王之女至上公主结婚。

5. 五子与诸盟友汇合。

##为什么是毗罗吒

黑公主被空竹骚扰，导致一系列结果，让俱卢诸位和阿周那提前相遇。

其中的隐藏信息是，毗罗吒王与三穴国敌对，三穴国是俱卢的盟友。

当时可以隐藏地点分为：

1. 婆罗门聚集地，太明显。

2. 五子盟友国，太明显。

3. 俱卢盟友国，太危险。

4. 中立国。

五子选择的时候并没有选择自己的盟友，也没有选择俱卢盟友，而是选择一个与俱卢间接敌对的中立小国。

即使如此，五子依然有暴露危险，如果怖军提前杀死空竹，那么三穴和俱卢提前攻打毗罗吒王，五子和阿周那面临两难选择。

要么继续伪装看着恩人国破，要么出战，没有遵守誓言。

##五子与毗罗吒联盟

毗罗吒是三穴宿敌，俱卢间接仇敌，投靠反俱卢方的五子的确是很好的政治投资。

这场婚礼盛大而幸福，即是一对新人的结合，也是五子和毗罗吒的结盟，五子盟友悉数到场，这婚礼盛大而欢喜，是十三年林居真正的结尾，流露着劫后余生的快乐。

婚礼上的人并没有想到，枝繁叶茂的般度一族会在战争中几乎绝后，激昂和至上公主的孩子成为唯一的继承人---继绝王。

#ifndef 林居篇总结.md

林居阶段故事脉络：

难受 => （阿周那升级武力）and（坚战云游圣地） => 香醉山重逢 => 回双林附近等待十三年结束。

 

林居经历从下面几个方面概括。

 

### 修行

 

阿周那提升战力和坚战云游圣地，都是在修行，从物理和精神两方面提升。

十三年林居，最大的转变是让五子的心智更加成熟。

坚战一路上听过无数的故事，为了救怖军身处险境。

阿周那在天国选择与兄弟复仇，而不是升上天国。

怖军在蟒蛇面前的自省。

都是逆境中才有的成长，这些光芒夺目的特质就藏在他们心中，但危难之中方才显现。

 

### 婆罗门

 

婆罗门在林居十三年中无微不至的照顾五子，在每个时间节点帮助和指点他们，也用渊博的比惨故事故事安抚他们，滋养他们的心灵。

 

### 盟友黑天和猛光

 

盟友黑天在十三年中，看望五子三次。

第一次是在林居第一年，第二次是在坚战云游途中，第三次在林居即将结束。

在五子整个林居过程中，黑天关心五子，为他们所受的苦难流泪，并拿起武器站在五子这边，支持坚战所做的决定，还和猛光一起帮他们抚养孩子。

 

### 意外

 

黑公主两次被劫，以及n次罗刹侵袭，都表明林居生涯非常危险。

强悍如怖军也在蟒蛇面前陷入危机，被坚战用机智问答救回。

五子林居中最大的危机发生在第十三年。掌管水源的药叉毒死四子。坚战通过机智问答救回四人。

进入香醉山的时候，遇到了自然灾难，黑公主晕倒。

### 生活条件

林居苦难，但是五子还是带了一些军队，仆役和厨师。

第一年林居，比较艰难，似乎没有厨师和仆役，坚战和黑公主一起准备饭。
