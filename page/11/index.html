#文明用语#侠 - 一个国产超级英雄的设定（二）

发表于 2015-06-15 更新于 2019-02-04 分类于 Text

第一篇设定：大概讲的是*文明用语*侠的定义，和打击对象。


本篇讲量刑规则：

*文明用语*侠以受害者所承受的伤害为量刑标准。

但是受害者所承受的伤害非常主观而且难以测量。

所以*文明用语*侠使用了以彼之道，还施彼身的手法。例如杀人者被杀。

这里又涉及了一个伦理问题。

以罪行惩罚罪行，如果是普通杀人伤害罪还好，但是强奸和轮奸以及其他性虐罪行该如何处理。

直接以彼之道，还施彼身，则*文明用语*侠本身就要成为强奸犯。（我们不介意超级英雄成为杀人犯，但是很介意ta成为强奸犯）

为了解决这个问题，*文明用语*侠制造了一种强奸模拟行刑器，模拟强奸中可能发生的行为，而且整个行为过程还会伴随：“你这个臭屌子，你是不是很爽啊，被干的很爽吧。”之类的精神侮辱，保证全360度代入，让加害者体验被害者的痛苦，实现刑罚准确。


为了模拟更多种犯罪，*文明用语*侠还制造了冷暴力模拟行刑器，侮辱歧视模拟行刑器，尽可能达到人间有多少种伤害，就有多少种惩罚。


文明用语侠---一个国产超级英雄的人设

发表于 2015-06-15 更新于 2019-02-04 分类于 Text

这位超级英雄直到现在都是个迷，因为天朝几个最大的媒体以及无数自媒体联合起来屏蔽它，在网上ta的名字打出来之后，再发布就是：*文明用语*。

*文明用语*的主要打击对象：

由于我国法律不够健全，所以ta的目标是法律没有惩罚或量刑太轻，而罪行又够重的那些人。

而ta通过受害者的痛苦程度来定义刑罚的程度。

例如：

强奸婴儿的（自称）残障患者，不久之后，大家发现他变成了真正的残障患者，而且肛门处塞入了印有*文明用语* logo的东北大玉米。

以及轮奸少女的若干xxx们，在不久后庭被插入多个印有*文明用语*的自动铅笔。

为了建立自己的工作安排，他还在网上新建了一个*文明用语*网，任何一个受到受伤害的人，都可以在上面写出自己或他人遭受的罪行，并且写出量刑参考。

谎报者将会在2-5个工作日内收到惊吓惩罚。

而查证成功（即使警方没有受理），也会在3-5个工作日得到相应惩罚。最高惩罚为死亡，而且是极为残忍非常痛苦而且极有创意的死亡，对渴望犯罪的人形成威慑。行刑工具上总会有*文明用语*的logo。


以罪行来惩罚罪行，以罪行来制止罪行，本身就是个挺难衡量的行为。

各类媒体并没有报道此事，甚至连谈论都禁止。因为谈论本身就是扩大*文明用语*的影响力。

而*文明用语*并没有惩罚这些屏蔽ta的媒体，只是在媒体以语言暴力伤害别人的时候才进行惩罚。


*文明用语*背后的人不得而知，但是在这个武侠泛滥的诡异年代，ta却是最接近侠客的人，因为ta倾听被害者的心声，将被害者的痛苦作为量刑唯一的标准。
