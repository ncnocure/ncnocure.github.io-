##难敌绝食

被俘之后，被宿敌救出，骄傲自大的难敌的三观动摇了，如果没檀那婆出来搅合，他的三观甚至会被摇回正常状态。

能被檀那婆以：

1. 毗湿摩等人被附体不再站在五子一边。

2. 迦尔纳会被骗走金甲，但加上敢死队可以赢阿周那。

这两个条件说服，证明蛋总并不是真正反省，改过自新，可能只是认识到自己和五子的实力差距实在太大，绝无取胜可能，所以宁愿死也不愿输。

后来他知道可能获胜的时候，就不再绝食。

##沙恭尼的态度

原著沙恭尼的态度一直是中立邪恶的。确定目标之后，可以不择手段，但并不刻意作恶。

这次他罕见的表明了不同的态度，让难敌报答五子，将王国归还。

采取这样态度的原因，是因为他站在难敌的角度思考，归还王国不仅可以平息难敌内心的惭愧，也可以为俱卢带来和平。

这也是化解矛盾最好时机。

波多黎各

一款工放游戏。

有四种数值：

分数和钱币，拓荒者，货物。

分数：用于判定胜负。通过货物出海，建筑，角色加成获得。

钱币：用于购买建筑，上面站拓荒者之后，建筑有效，可以获得分数或者钱币上加成，以及其他功能。

货物：用于出海换取分数，也可以在商人回合卖掉获得分数。通过获取种植园并让种植园有效之后，进行监管回合获得货物。

拓荒者：用于让购买的建筑和种植园有效。通过市长回合获得。

 

游戏流程是：

每个局，每个玩家依次选择7种身份。每个身份自身包含特权，还可以让所有人进行一个行动阶段。

例如有玩家选择建筑师，可以让所有人进入建筑建造阶段，可以选择一个建筑建造。

每个行动阶段的关系图如下：

 

游戏的趣味点在于，建筑和种植园的建造和规划。

 

电厂

四个元素：

1. 电，用电厂花费资源获得。

2. 城市，花钱修通管道，占据地图点获得。

3. 钱，发电获得。

4. 资源，购买获得。

关系如下：




每回合每人依次四种行为：

1. 拍卖电厂。//拍卖之间的嘴炮和欺骗

2. 购买资源。//炒高资源，获得收益

3. 占据城市。//封对手的路，让对手无路可走

4. 发电。玩家对自己占据的城市，通过使用电厂所要求资源进行发电，得到钱。//尽量用便宜电厂发最多的电，根据不同局面，需要权衡钱和浪费。

值得注意的是，1，2，3种行动都需要花费金钱，挣钱的方式只有4.规划1，2，3各自花费的钱就非常重要。

有时候拍卖阶段花钱太多，会造成占据城市时被压制。

为了撸mad，反复反复的看片还是很有收获。

1. 男主虽然没明显表情，但是反复拖视频的时候，感觉他回头非常有力，这个动作似乎就代表喜悦和惊讶。

2. 女主的白色蕾丝似乎代表贞洁/亲情以及一系列的东西。当她用白沙挡住面容的时候，是她还在羞涩的拒绝。当她手中线断掉，意味着她已经愿意放弃这些。当她摘下带着线的戒指，意味着她已经下定决心。后面的红纱代表感情，黑纱代表牺牲。被爱情所动，而后冒着死亡的危险在一起。

3.全篇缓慢的动作让仪式感非常强烈。尽管为了压节奏，不得不把几乎每个片段都调快，但是反复拖动之后，我还是喜欢原版的缓慢，镇定而沉默。

4. 最后爱人埋葬爱的部分我最为喜欢，男主举起诗句，漫天纷飞的骨灰，即使在过去的拍摄条件下依然美丽无比。而他埋葬不是爱人的尸体，而是他夭折感情，从此人之爱在他生命中褪去，由神之爱取而代之，即使爱人找回他，爱却无法复活。

5. 这部片因为元素非常少，很多画面就是一个角色在画面中央缓慢的凸造型，所以你反而能一帧一帧的研究他们每一个动作，每一个表情，回头过程中，从何时眼睛带上了喜悦。以及双手垂下的时候，何时变得无力任由情爱控制。这一系列的细节过多，不好解读，但是看懂之后觉得很美。

TheGamers Con.桌游极客大会是桌游资讯媒体TheGamers举办的桌游玩家聚会。

这次大会的整体感受如下：

1. 桌游种类极为丰富，从简单容易的聚会类，到重度策略类和类万智，以及强代入感的跑团，还有桌面冰壶之类的桌上运动。可能因为桌游群体较为小众精英，玩法需求也更加多样化。

2. 爱好者男女比例持平，女性玩家比想象中多，包括重度桌游爱好者，但更偏重聚会类和协作类游戏。

3. 卡牌更加注重包装，与ACG等热门元素结合。

4. 桌游群体更加社团化，即每个到来的玩家几乎都是属于某个日常认识并长期面对面的稳定桌游团体。桌游因为面对面游戏的特点，导致其社交团体非常稳定。

1. 塔希里亚系列： 国产游戏里最有代表性原创ip。其中塔希里亚法师是接龙类游戏，可惜有bug，体验并不好。诡法牌的规则极为复杂，但做工非常精美，即使不作为桌游，作为成套手办也是非常棒的选择。

 

正义没有玩，粗略看了一下是一款战略类游戏。

2. 谁知我心：MOOD X

这款游戏是整个体验过程中，个人认为最适合聚会社交的游戏

1. 讲故事。

2. 揣摩对方心意。

 以4人局为例，玩家分为A，B，C，D。流程如下：

A抽取三张卡片，从中选择一张，卡片类似下图，以两个角色表述一个故事。



B从卡片上指出一个主角。以上图为例，指刀指着即将跳海的人为主角。

C需要以即将跳海的人为主角，根据卡片上图，编出一个故事。例如一名歌手被海盗绑上船，他在临死前希望海盗能让自己唱最后一只歌。海盗同意了，一边欣赏着他的歌声，一边用刀子逼着他往木板边缘移动。

D需要根据这个故事，在心情板块上选择此刻的心情。心情板，由八个图片组成，每个图片上并没有标明心情，需要玩家根据不同语境来理解图片上角色的表情。例如同样的图片可以代表快乐也可以代表释然轻松。

然后其他人使用类似的心情盘，猜D选择的心情。 

结果有如下几种情况：

1. 有人猜中D的心情，则猜中者和D一起得分。

2. 没人猜中D的心情，则所有人得分，D不得分。 

所以D想要获胜必须选择只有少数人能够猜中，但又不是所有人都能猜中的心情。

也就是他的心情必须能够猜到，但又出人意料。

例如之前的小故事：一名歌手被海盗绑上船，他在临死前希望海盗能让自己唱最后一支歌。海盗同意了，一边欣赏着他的歌声，一边用刀子逼着他往木板边缘移动。

D可以选择两种心情，恐惧和释然。

面对死亡的恐惧和临死前天鹅之歌的释然两种都可以解释通。

选择恐惧，会让很多人猜中。

选择释然，则有人能够猜到，但也需要承担风险。

选择其他不着调的心情，则被猜中可能性非常小。

 

对于猜心情的人来说，也是个“我知道你知道我所想”的循环困境。

选择恐惧，则对方很有可能避免这种所有人都能猜中的情况。

选择释然，则对方也可能选择更加稳妥的恐惧。 

一回合结束之后，A，B，C，D四人轮换选卡牌，选主角，编故事，指心情的角色。为了避免一个人说故事，总由固定的人指心情，还可以随机打乱。

3. 强权外交



 嘴炮，战棋。

规则极为简单，但是过程极为有趣。

以前玩过的冰与火之歌是该款游戏的复杂衍生版本。

 因为没有凑够人，所以只体验其中一个四人简化版本。

该游戏与其他战棋策略游戏不同的地方在于：

1. 游戏规则很少，鼓励玩家不择手段。可以随时结盟和背盟，任何盟约仅为口头，并没有任何约束效果。

2. 没有先手后手之分，所有人在纸上写指令，由gm来结算。 

游戏整个过程中，战棋推演运算相对简单，但是彼此结盟，防止背盟，制衡对手，选择策略非常复杂。

4. 锦树

一款简单而美丽的拼图类游戏。类俄罗斯方块。有趣之处在于每次完成游戏，都能得到一幅美丽的画。 

5.唐伯虎点秋香聚会类桌游，本次桌游展少有的失败作品。

1. 目标模糊，玩家无法建立行为—结果的映射，即使是聪明的玩家不知道自己行为的效果，所以也没法选择行为，让整个游戏过程呈现一片茫然。

3. 角色不平衡，阵营分化过多，有些角色非常难以获胜。 

这款游戏试图使用唐伯虎点秋香的经典ip构建角色，试图给玩家更强代入感，从剧情出发来思考技能，而并没有明确玩家游戏过程中的目标和行为。玩家不知道怎样赢，甚至不知道怎样输。 

6. 牧场物语 

这是一个游戏玩家自己diy的游戏。涉及资源管理和策略。

每个玩家拥有一个资源栏和牲畜栏。可以通过放置棋子进行收获，喂养牲畜和放置牧场。最终的分数，与玩家手头的牲畜和牧场数和收获次数有关。没有任何包装，游戏规则也很朴素，但是游戏已经具备基本的策略管理游戏的雏形。 

7. 追女生

通过5种元素追求女生，并带走女生的故事。本质是投资游戏。随机获得五种资源，或投资不同的女生，或抄高资源价值，价值最高者获得女生。

 以上是桌游展会中印象较深的游戏。

#文明用语#侠 - 一个国产超级英雄的设定（二）

发表于 2015-06-15 更新于 2019-02-04 分类于 Text

第一篇设定：大概讲的是*文明用语*侠的定义，和打击对象。


本篇讲量刑规则：

*文明用语*侠以受害者所承受的伤害为量刑标准。

但是受害者所承受的伤害非常主观而且难以测量。

所以*文明用语*侠使用了以彼之道，还施彼身的手法。例如杀人者被杀。

这里又涉及了一个伦理问题。

以罪行惩罚罪行，如果是普通杀人伤害罪还好，但是强奸和轮奸以及其他性虐罪行该如何处理。

直接以彼之道，还施彼身，则*文明用语*侠本身就要成为强奸犯。（我们不介意超级英雄成为杀人犯，但是很介意ta成为强奸犯）

为了解决这个问题，*文明用语*侠制造了一种强奸模拟行刑器，模拟强奸中可能发生的行为，而且整个行为过程还会伴随：“你这个臭屌子，你是不是很爽啊，被干的很爽吧。”之类的精神侮辱，保证全360度代入，让加害者体验被害者的痛苦，实现刑罚准确。


为了模拟更多种犯罪，*文明用语*侠还制造了冷暴力模拟行刑器，侮辱歧视模拟行刑器，尽可能达到人间有多少种伤害，就有多少种惩罚。
