##粉黑大战改变历史进程

本章中毗湿摩与童护大段大段争论。

争论的焦点在于：

毗湿摩认为黑天是世界主宰。

童护不信。

如果毗湿摩单纯是在王祭上维护本方德高望重的盟友，就不必神话黑天。

但本章中，毗湿摩已经变成黑天的信徒，他与童护的争论，变成了黑天信徒与不信者的宗教争辩。

宗教和政治斗争掺杂在一起，生硬掺杂，毗湿摩也彻底OOC成黑天脑残粉。让本章充满着weibo/tieba上粉黑之争的奇妙氛围。

【脑洞】我猜本章节辩论中的宗教因素，很可能是后天加上的，而且加的非常生硬。下面闲扯就忽略宗教因素了。

##童护攻击黑天和毗湿摩的动机

撕逼大战的导火索是首席献礼。

童护借首席献礼，攻击毗湿摩，即攻击毗湿摩背后的五子，挑起争端，破坏王祭。

童护选择这样的时机对五子发难，值得玩味。

在妖连死后，童护作为妖连旗下统帅，没有为妖连报仇；征四方过程中，童护亲自出城迎接怖军，按照坚战的意愿上缴税金。

'于是， 怖 军 把 法 王 坚 战 的 意愿 对 他说 了， 国王 童 护 答应 照办。国王 啊！ 怖 军 在 童 护 那里 住了 十 三夜， 备受 尊敬， 然后 带着 大军 和 车 骑， 继续 前进。'

童护带着儿子参加王祭。

在黑天接受献礼之前，童护与五子的关系友好。

黑天接受首席献礼事件，为什么会让童护突然发难。

童护向五子发难有两种动机：

1. 嫉妒黑天，单纯情绪。

2. 政治诉求。

注意：人做一件事情的时候，往往多个动机掺杂。政治和情绪可能一起导致童护的行为。

if (童护向五子发难是政治诉求)

{

    王祭上可以获得首席献礼的人，在政治意义上就是五子一方地位最高的政治盟友。

    黑天接受首席献礼名至实归：

    {

        整个五子崛起过程，离不开黑天的支持：

        虽然甘蔗王是五子的姻亲，但黑天才是五子最依赖的盟友，如父亲一样看护般度族成长壮大。

    }

}//if (童护向五子发难是政治诉求)

童护能够煽动成功，是因为王祭上的刹帝利并非铁板一块。

##童护背后的势力

此时王祭上刹帝利分为下面几派：

1. 妖连旧部：

以童护为首。

童护本是妖连旗下大将，妖连虽死，但是妖连旧部依旧，妖连以武力和政治建立起的联盟还有希望复活。

2. 畏惧/嫉妒五子崛起之人：

这群人与妖连关系较远，并不会为妖连之死而反抗五子。童护带头挑战五子权威之后，他们出于内心本就存在的嫉妒和畏惧也会站在童护这边反对五子。

难敌也属于这种人，碍于俱卢和五子的交情，他没有出声。

3. 易被煽动者

中立者中不清醒的人，很容易被童护富于煽动力的话语左右。

“黑天不是国王，黑天杀死自己的恩人，黑天不配享受最高献礼，赋予他最高献礼的人徇私，这些人举办的王祭犹如儿戏。”

不患寡患不均，唤起公众对不公的愤怒，就让一群没有利益关系的人头脑发热的站在童护这边。

4. 中立者

这群人不说话，他们也有态度，但不表态。

5. 泛五子派

这群人主要包括，木柱王系，多门城系以及象城那边五子真正的亲友，还有征四方中被五子真正征服转化为盟友的一群人。

童护的挑衅对三种人中的1，2，3起作用，他通过挑衅毗湿摩，挑起五子俱卢与在座国王们的矛盾之后，开始攻击黑天。

黑天有誓言不杀童护。

若童护凭借誓言庇护，有恃无恐的当众挑战五子最珍视的盟友，没有遭到任何打击报复。

1，2，3三种人就知道挑战五子，挑战王祭不会受到惩罚，他们心中最后一点顾忌就会消失，甚至会把王祭变成群殴现场。

王祭遭到破坏，五子和黑天的声望将一落千丈，他们刚刚通过征四方建立起的联盟会崩溃。

到时童护可兴兵攻击势单力孤的五子，获得威望，成为妖连联盟的继承人。

这一切建立在黑天守誓的基础上，然而他是神，超越誓言的存在。

黑天用温和的语气陈述罪行之后，秒掉童护，而后手持武器，以武力威慑在场1，2，3种人，让王祭正常进行。

##黑天的誓言

'即使你的儿子有一百条罪过，应该被杀，我也会饶恕他。'

到底童护有没有集齐100+1条罪过，黑天杀童护算不算破誓。

一直对誓言非常看重的印度神话此时模棱两可。没说是也没说非。

if (黑天杀童护 == 破誓)

{

    在摩诃婆罗多的世界里，黑天是第一个敢于不守誓言的人。

    我信息有限，不知道印度刹帝利和婆罗门发誓必守誓的传统来自何方。如果发誓不守誓会承担怎样的惩罚。

    目前来看，誓言虽然是形式，但终归源于人心，例如毗湿摩的誓言源于他对父亲的爱和对象城的责任。

    黑天的誓言源自他对姑母的爱。当黑天违反誓言杀死童护的时候，他要面对不只是上天的惩罚，而是自己姑母的眼泪。

    但破誓的代价与王祭被破坏比起来，孰轻孰重。衡量之后，他做出了选择，破誓杀掉童护。

    无论他是人，是神，在誓言面前做出选择，都是智慧和勇气的举动。

}//if (黑天杀童护==破誓)

{

    所以婆罗门编出神话，将他变成三相神的化身。

}
