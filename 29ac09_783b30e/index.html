        五子住在树林里，以野果为食，坚战为赌博而痛苦，如心中长刺。林居十三年结尾将至，大家心境有所变化。

        此时毗耶裟到来安慰坚战，两人谈起施舍和苦行哪种更重要。

        {

            一位拾落穗为生的苦行者，举行名为希望的祭祀。用米供奉天神和客人，用剩下的米维持自己的身体。

            以天为衣的杜婆沙吃光了苦行者辛苦得来的所有食物，一连六次，而苦行者心境毫无变化。

            杜婆沙让苦行者肉身升入天国，然而苦行者得知“业果耗尽之后，将从天国坠落”，于是他没有选择天国，而是达到永恒和至高的成就，名为“涅槃”。

        }

        毗耶裟用这个故事安慰坚战，消除他内心的恐惧。

    }

##五子此时的心境

 

林居十三年结尾将至，大家心境有所变化。

坚战：总感觉兄弟们承受巨大的痛苦，都是由他的罪过造成的。想起自己的罪过，他就不能安睡，心头上仿佛扎着刺。

其他人：想到流放时间已剩下不多，勇气和怒气使他们激动得几乎变形。

 

毗耶娑想到这点，前来安慰他们。婆罗门对五子的照顾真是无微不至。
