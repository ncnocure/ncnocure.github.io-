#14-维杜罗来临篇# and #15-得国篇# 札记（四大老狐狸尽数登场）

发表于 2015-03-03 更新于 2019-02-08 分类于 Text

#ifndef  14-维杜罗来临篇 and 15-得国篇

 

//14-维杜罗来临这个章节简述了象城各派势力对“五子娶黑公主”的反应，角色特点鲜明。

//如果你想要说服别人，维杜罗的发言值得再三阅读。

 厉声：

{

    //维杜罗来临篇开始

    护民子讲述各方势力的反馈：

    {

        各国国王通过密探得到消息：黑公主嫁给五子。

        难敌设置诡计被揭穿，国王们谴责持国和毗湿摩。//五子得到木柱王的支持，成为冉冉新星，此时谴责象城，就是向五子示好

        难敌愤怒，维杜罗喜悦，持国则表达对“五子结盟木柱王”喜悦。

        难敌对持国的“喜悦”感到愤怒。

        持国：'“我的想法正和你们的一样，不过我不愿在维杜罗面前披露。所以我特地夸赞他们的优点，好使维杜罗从表情上看不出我的真意。难敌，迦尔纳！你认为现在该怎么做？”'

        难敌：

            1. 离间贡蒂之子和玛德利之子之间关系。

            2. 离间木柱王和五子

            3. 离间五子和黑公主

            4. 悄悄把怖军弄死

            5. 派遣女子引诱五子，让黑公主不满

        迦尔纳先对难敌几条计谋逐一提出反驳：

            1. 离间五子----他们爱着同一位妻子，不会被分裂。

            2. 离间黑公主和五子----黑公主在五子落魄时选择他们，不会嫌弃他们

            3. 离间木柱王和五子----木柱王和猛光都不贪财，不被引诱。 

        迦尔纳的建议：

        {

            趁五子根基尚未稳固，消灭他们。

        }

        持国让迦尔纳与毗湿摩，德罗纳和维杜罗商量。

        //象城三大实权派不会出兵攻打五子

        //即持国已知迦尔纳的建议毫无可行性，但并不亲自否定他，而是让他与持反对意见的实权派商量，借刀杀人借助迦尔纳抗衡象城实权派，自己不出头

    }

    象城开会讨论如何应对“五子结盟木柱王”：

    {

        毗湿摩反对出兵五子，建议难敌将一半国家还给五子，表达充分善意。//毗湿摩的出发点是正法和道义

        德罗纳同意“将一半国家还给五子”，并建议打亲情牌，与五子保持良好关系。//德罗纳关注细节

        迦尔纳认为德罗纳和毗湿摩偏向五子，没有为持国考虑。//迦尔纳一直站在难敌这边

        德罗纳：你仇恨般度族，才指责我们。不听我们的建议，俱卢族就会毁灭。//不是危言耸听

        维杜罗：毗湿摩和德罗纳德才兼备，在持国族和般度族中间保持中立，提出的建议对您有利。

             五子实力强大，拥有黑天和木柱两位强大盟友，城内居民也支持五子，你打不赢他们，请用亲善怀柔的方式与般度族结盟，并且洗刷紫胶宫的污点。//维杜罗以持国利益作为论述出发点

        持国认可维杜罗等人的建议，派遣维杜罗对五子表达善意，并邀请五子与黑公主回象城。

    }

    //15-得国篇开始

    护民子说：

    {

        般度五子在黑天和维杜罗的陪伴下，回归象城从持国手中获得一半国土，搬到甘味林居住。

        在黑天的引导下，般度五子建立布局合理富丽堂皇的天帝城。

    }

}

#endif

 维杜罗来临篇表达了几方面势力对“五子娶黑公主”事件的态度。

1. 各国国王向五子示好

2. 难敌：更加仇恨五子，懊恼

3. 持国表面表达善意，实质和难敌一样

4. 维杜罗，毗湿摩，德罗纳等人喜悦

 象城诸位对五子下一步行动：

{

    难敌：离间五子，黑公主，木柱王。

    迦尔纳：趁其根基未稳攻打五子。

    毗湿摩：从道义出发，劝持国将半个国土还给五子

    德罗纳：同意毗湿摩建议，从细节上向五子示好

    维杜罗：从持国自己的利益出发，说服持国与五子结盟。

    持国：以自己和难敌的利益为主，听取有利自己的建议。

}

毗湿摩，德罗纳，维杜罗三人的建议相似，但是出发点和表达方式完全不同。

毗湿摩专注道义：

    1. 般度五子有权得到半个国家，他们被不义的赶出王国，应该得到分内的半个王国。

    2. '自从我听说贡蒂死后，在这世上，我就无颜见任何活人了。'//准确点说毗湿摩专注个人道义，五子被害让他感觉惭愧，得知五子没死，他非常高兴，他忍不住要用一半国土补偿五子。

德罗纳叙述向五子示好的具体细节。毫无根基的德罗纳只是个顾问，对于决策没有发言权，但可以让细节尽善尽美。

维杜罗论述利益：

    我的建议对你有利。我知道你心里偏袒难敌，但是与五子不合，对你和难敌都没有好处。

    因为五子本身强大，有强大的盟友，不能被战胜。

    谋害五子的污点会让你荣誉受损，也会破坏你和五子之间的关系。 

    与一直为敌的木柱王和人多势众的黑天结盟可以增强象城实力。

    国民和舆论支持五子，而不支持难敌和迦尔纳，沙恭尼。

    综上为了您自己的利益，请与五子结盟。//全篇没有提到分一半国土给五子，而是说'以适当的态度来对待五子'，先确定持国对五子的态度。

    //国土问题敏感而充满争议，如果太早提出反而会分散讨论者的注意力

 持国的态度最暧昧。

私下里，面对难敌：'“我的想法正和你们的一样，不过我不愿在维杜罗面前披露。所以我特地夸赞他们的优点，好使维杜罗从表情上看不出我的真意。”'

大会上，他接受维杜罗的建议，并派遣维杜罗迎接五子和黑公主回国。

 持国的真实立场，在面对难敌时表露：

他和难敌一样，将五子视为象城继承权的有力争夺者。

而在大会上，持国又被维杜罗说服与五子结盟，因为这个决策最符合难敌方的利益。

这证明持国的智慧和自我控制不亚于维杜罗，黑天，坚战几大狐狸。 

目前的难敌方，沙恭尼在原著中是赌博专业人才，赌局之前没正式发挥作用。迦尔纳是一名战士，思维直线。

而此时的难敌其实不是个很好的政治家。因他不善忍耐，若由着他胡作非为，难敌方大概很早就灭了。

目前持国是难敌方真正的核心，这位盲人惯于忍耐，善于隐藏，精于妥协，借刀杀人，很少自己出头，符合一个高级政客的要求。

他宠爱儿子，但又知道顺应儿子的想法不一定能达到有益的结果。他知道天命，甚至意识到势不在我，却又没法改变。


 维杜罗能看透他，所以以难敌利益为出发点来说服他，而他也能看透维杜罗，这两个人交锋真有趣。


至此摩诃婆罗多四大老狐狸持国，维杜罗，黑天，毗耶娑尽数登场，请尽情欣赏人精与人精之间争锋相对。


目前坚战只能算小狐狸。难敌除了在迦尔纳事件上占得先机，其他时候基本处于下风，更像豺狼。赌神沙恭尼目前还没正式亮相。


//毗耶娑对这场会议要么是亲眼所见，要么是听在场人转述，因为每个人的发言都从自己立场出发，富有个人特点而滴水不漏。


 ##得国篇里的黑天


撕逼大战已经在14-维杜罗来临篇进行过了，象城已经在对待五子的态度上达成共识。得国篇里一切都顺理成章，也没什么好说的。

只有黑天的出现，略有异样。

黑天和般度五子一起回象城，并五子得到一半国土之后，帮他们建立天帝城。

建立一个城市（还是首都）需要考虑的事情非常多。

    1. 防守外敌

    2. 功能区划分

    3. 道路排布

    4. 排水，防火等基础建设

黑天无私的奉献自己的智慧，经验以及财富，如果不考虑他是天神下凡，推动天命，仅以黑天作为凡人的立场来考虑，则是黑天欣赏五子，相信五子会崛起，于是在五子身上投资，让般度族这系政治势力发展壮大。

之后系列篇章中，这笔投资得到丰厚的回报。

 

//黑天与五子的亲缘关系：贡蒂是耶度王苏罗的女儿，及黑天之父富天的妹妹，黑天的姑姑。黑天和五子是表兄弟关系。这是黑天帮助五子的另一层原因。

 

伴随得国篇结束，五子终于拥有自己的国家，在群雄并起的古印度版图有了一席之地，般度之子的好运远未到头，他们将修建举世无双的大会堂，杀死妖连，征服四方，举办王祭，直到王祭之后的赌博，命运之线才骤然坠落。
