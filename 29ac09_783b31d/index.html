#摩诃婆罗多# 44-引火木篇 札记（四个不省心的弟弟。。。）

发表于 2015-07-04 更新于 2019-02-02 分类于 Text

#ifndef 44-引火木篇

//四个不省心的弟弟。。。

 

厉声说：

{

    护民子说：

    {

        五子和黑公主离开伽摩耶加树林，回到双林。

        一位婆罗门被一条鹿带走了引火棍和引火木。

        五子为了寻回引火木追着鹿，身心疲惫。

        一棵树下，五人回忆起之前的苦难。

        无种：我们遵循正法，在众生中无与伦比，为何还会遇到麻烦。

        坚战：正法之神按照善恶分配不幸。

        怖军：当初我没有杀死把黑公主当作奴仆带进大会的使者，让我们遭到麻烦。//怖军对黑公主的受辱，真是耿耿于怀。

        阿周那：我容忍迦尔纳说出粗言恶语，让我们陷入麻烦。

        偕天：我没有杀了沙恭尼，让我们陷入麻烦。

 

        此时五人非常口渴。

        无种找水，水池旁，一个声音说：先回答问题，再喝我的水。

        无种不听劝告，倒下、

        偕天找无种，一个声音说：先回答问题，再喝我的水。

        偕天不听劝告，倒下。

        阿周那找两个弟弟，一个声音说：先回答问题，再喝我的水。

        阿周那：你站出来阻止我，让我用箭射穿你！

        说着阿周那射出箭雨，覆盖整个区域，显示他射声（听声辩位）的本领。//23333333

        然而并没有什么卵用。

        阿周那口渴难耐喝了水之后倒下了。

        怖军来找三个弟弟，他看到三个弟弟都倒下了，以为是药叉或罗刹干的，心想：“今天我肯定要打一仗了，让我先喝点水。”于是他喝完水也倒下了。//23333333

        坚战来找四个弟弟，看到四具尸体。他回答了药叉的所有问题，得到药叉的恩惠。

        药叉：我可以复活你一个兄弟。

        坚战：复活无种吧。

        药叉：'你喜欢怖军，依靠阿周那，为什么要让庶出的无种复活呢？'

        坚战：玛德利也是我的母亲，所以让她的一个孩子复活吧。

        药叉：你认为仁慈比财富和爱欲更高。让你所有兄弟复活吧！

        这个药叉并不是普通人，他是坚战的父亲法王阎摩。

        婆罗门丢引火棍，以及喝了会死的水，都是阎摩为坚战设下的考验。

        药叉给了坚战恩惠，让他们在第十三年内隐匿生活不会被人发现。

        而后五子为第十三年的隐藏乔装做准备。

    }

}

 

 

#endif

##主要内容

正法神为坚战设下考验。

让他四个兄弟死去，再让他选择一人复活。

坚战既没有出于个人感情复活最喜欢的怖军，也没有出于财富复活最有用的阿周那，而且出于仁慈复活玛德利的儿子无种。

 

##阿周那和坚战的反映都好萌。。。
