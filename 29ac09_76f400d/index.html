厉声说：

{

    护民子说：

    {

        五子等人前往伽摩耶加林。黑天和大苦行者摩根德耶来到此地与五子相见。

        黑天：

        {

            恭喜阿周那得到武器。告诉黑公主她孩子们在多门城过的非常好。//黑天真温柔。

            对坚战说，我们是你的盟友，带领军队完成你的事业。

        }

        坚战感谢黑天。

        摩根德耶到来开始讲故事：

        {

            第一段关于业：

            {

                最先出生的人，与梵同一，没有烦恼，可以上天与天神相聚，不为生死所困。

                后来的人，受爱欲和嗔怒控制，被天神遗弃，寿命短促，得恶报。

                业决定人死后的归宿。人的身体死后，立即转生，所做的业如影随行。

                有人今生幸福，有人来世幸福，有人既有今生又有来世，还有人今生来世都不幸福。

                而你们拥有勇气和经历，又能控制自我，取悦天神。遇到麻烦时不要怀疑，你能享受幸福。

            }

            婆罗门伟大品性的故事：

            {

                海诃夜族的王子游荡时，打猎，误杀一名牟尼。

                当他向牟尼的父亲道歉时，得知那位死去牟尼已经复活。

                恪守正法，善待婆罗门让他复活。

            }

            另一个婆罗门伟大品性的故事：

            {

                大苦行者阿多利想去森林生活，临去之前，他访问一位正在举办王祭的国王，试图得到钱财分给自己的儿子们。

                王祭现场，他赞颂国王：“你是大地之主，除你之外没人懂得正法。”

                另一位仙人乔达摩认为生主因陀罗占据首位。

                两人一次撕了起来。

                知法的永童来解除两者疑问：

                这涉及梵与王权的关系。认为国王至高无伤的一方取胜。

                国王非常高兴，赐予阿多利财富。阿多利将财富分给儿子，自己去森林修行。

            }

            多尔刹和辩才天女之间对话，涉及：

            {

                1. 如何不偏离正法。

                2. 古代火祭的戒规。

                3. 牟尼达到最高解脱是什么？

            }

            毗婆薮之子摩奴的故事：

            {

                摩奴为太阳神毗婆薮的儿子，常年修炼苦行。

                一日他长发盘头修行时遇到一条小鱼。 小鱼祈求庇护。

                摩奴收留小鱼。小鱼不断长大水罐到水塘，再到恒河都盛不下它。

                摩奴将它放入大海。

                大鱼：洪水时代即将到来，你要让人为你造船，偕同七位仙人登船，带上各式各样的种子。你要在船上等我回来。我头上有犄角，你会辨认出来。

                洪水到来，摩奴遵循鱼的指示登船。一条带犄角的鱼出现，摩奴把缆绳挂在它的犄角上。

                鱼将船拉到雪山山峰。

                原来鱼是大梵天，来解救摩奴。

                而摩奴是芸芸众生的创造者。

            }//这篇是《鱼往世书》，实在是太萌了

            摩根德耶见过几千个时代的末日，他讲述每个时代的过程：

            {

                世界毁灭之后，一切奇迹从头开始。

                圆满时代持续四千年，开始和结束各占四百年。

                三分时代持续三千年，开始和结束各占三百年。

                二份时代持续两千年，开始和结束各占二百年。

                争斗时代持续一千年，开始和结束各占一百年。

                争斗时代人人说假话，婆罗门做首陀罗的事，首陀罗则按照刹帝利方士生活。

                毁灭之火借助狂风燃遍世界，焚烧一切。

                然后暴雨淹没大地和火焰，四处狂风大作，自在天吸进可怕的狂风。

                那时候只有我一个人活着，在这片汪洋中孤独游荡。有一天我在榕树上看到一个眼似莲花，胸有卷毛的孩子。

                孩子说：我知道你很疲乏，想休息一下。如果你愿意就住在这里吧。

                孩子张开大嘴，我身不由己进入他的嘴中。

                里面布满王国和城市，河流山川大海尽在其中，因陀罗为首的天神等种族也在他肚子里，简直是另一个世界没有尽头。

                我祈求至高神的庇护。又离开他的口中，他坐在榕树上拿着整个世界。

                孩子：休息的好吗？

                我向他行触足礼，问：为什么你在这里显现为儿童，吞下整个世界。

                孩子向摩根德耶讲述自己和世界和众生的关系，讲完之后大神消失。

            }

            我见到你亲戚黑天，他就是那位眼似莲花的大神。你们向这位救主寻求庇护吧。

            听完，五子和黑公主一起向黑天致敬。

            摩根德耶讲述非法时代之后的事情：

            {

                先简述了非法时代的混乱而残暴情景。//非常像现代。其中种姓将不存在，人们甘愿追随首陀罗。

                喧嚣和混乱中，时代毁灭。世界以婆罗门为首渐渐回复正常。时神催促下，一名叫做加尔基毗湿奴耶舍的婆罗门诞生在商勃罗村，他将以法取胜，成为转轮王，引导混乱世界走向稳定，结束这场毁灭。

                于是非法灭亡，正法昌盛，圆满时代到来，进行下一次循环。

            }

            摩根德耶总结：任何时代都不要对婆罗门傲慢无礼。

            一个关于婆罗门高贵的小段子：

            {

                国王与青蛙之女生下三个儿子，大儿子舍罗即位。

                舍罗侵占仙人的马匹不还。仙人诅咒他被铁叉杀死。话音刚落，手持铁叉罗刹出现杀死国王。

                第二个儿子陀罗即位，依然不还马，还想杀死仙人。

                仙人用诅咒杀死陀罗的十岁的儿子。陀罗无法攻击仙人，认输结束了这场灾难。

                仙人给予王后一个恩惠，并拿回马匹。

            }

            论名誉的重要性：

            {

                一位王仙公德耗尽之后，从天而降，被认为名声灭绝。

                王仙找到一直猫头鹰，然而猫头鹰不记得他。

                而后他找到一只比猫头鹰更老的仙鹤，仙鹤不记得他。

                他又找到比仙鹤更老的乌龟，然而这只乌龟记得他，记得他曾一千次将乌龟放在祭坛上。

                王仙重归天国。善业的声誉响彻天地，这个人就能受到称道。

            }

            尤腾伽帮助巨马之子杀死敦杜：

            {

                尤腾伽以苦修得到毗湿奴恩赐：协助巨马之子杀死敦杜成为敦杜罗摩。

                尤腾伽劝说甘蔗王后裔巨马之子杀欲毁灭世界的敦杜。

                提迭敦度的来历：

                {

                    两个檀那婆见到毗湿奴。他们过于高傲，让毗湿奴接受自己的恩赐。

                    毗湿奴求的恩赐是：'让我杀死你们，为了世界的利益，我希望实现这个愿望。'

                    两个檀那婆同意，希望死在没有遮挡的地方，同时请求成为毗湿奴的儿子。

                    两人的儿子叫敦杜。//他们倆生子了。。。

                    敦杜修行大苦行得到梵天恩赐，让除人以外的种族都无法杀死自己。//除人意外种族无法杀死自己，这梗用了太多次。。。

                    敦杜遂天下无敌。

                }

                尤腾伽将那罗延之力注入巨马之子体内，让他带领儿子杀死敦杜，自此巨马之子以敦杜罗摩闻名于世。

            }

            摩根德耶：为忠于丈夫的妇女制定的正法。

            {

                雌鹤将粪便排泄在婆罗门头上，被婆罗门当场咒死。婆罗门又后悔，认为雌鹤罪不至死。

                婆罗门来到一户人家里，女主人为了照顾父亲不慎怠慢婆罗门。

                女主人以雌鹤为由解释自己的行为，并指引婆罗门到密提罗寻访一位猎人。

                婆罗门原谅女主人，又好奇女主人知道雌鹤的事情。

                他来到密提罗寻访那位猎人。猎人与婆罗门对话，告诉他：正法与业，控制感官，人的暗忧善三种属性。

                借猎人的口再次提出行为论：

                '一个首陀罗出身的人，如果奉行美德，就能成为吠舍，也能成为刹帝利。如果他行为正直还可以成为婆罗门。'

                最后猎人劝婆罗门回去侍奉父母，婆罗门同意认为自己被挽救了。

                而这位猎人前世也是婆罗门，因为杀死一个牟尼而被诅咒成为首陀罗的猎人。

                仙人告诉他：孝敬父母就能获得巨大成就，升入天国。

                由此再次说明行为决定种姓，错误行为导致不幸遭遇。

            }

            摩根德耶火的世系：//这里是五个世系将自己的修行诞生一人五生，五生生出诸位天神和众生。

            //五生和鸯耆罗后裔又生出无数其他种类的火

            {

                火神修炼苦行。

                鸯耆罗将自己变成火神照亮整个世界。

                鸯耆罗让火神赶紧回复地位。

                火神：我的声誉已经丧失。

                鸯耆罗：你做让众生升入天国的善事吧，我成为你第一个儿子。//鸯耆罗是梵天第三个儿子，也称祭主，鸯耆罗其中一个儿子是毗诃波提，天神之师，云发的父亲

                鸯耆罗生有女儿和儿子，代表各种火焰：

                祭火，不落之火，胜世之火，享世之火，享上之货，如意之火，爱欲之火，不空之火，歌颂之火，发怒之火。

                迦叶波，极裕，波罗那，波罗那之子，鸯耆罗之火，行落和三辉经过苦行求得一个儿子，名为五生，是五大世系的奠基人。//大约是钢铁侠，托尔，贾维斯和奥创一起生幻视的感觉

                五生修炼万年苦行，创造了湿婆，因陀罗，风和火，又创造了世界众生。

                有创造了祖先的五个儿子，毗诃杜尔阁的儿子，迦叶波的儿子，鸯耆罗的儿子，婆尔遮的儿子，波罗那的儿子，他们是五个世系的后裔。

                他还创造了众天神，以及骚扰祭祀天国邪神。

                婆罗多火受到严格约束，又名满意之火。

                湿婆火拯救一切受痛苦折磨的人，所以被称为湿婆火。

                大地之主与天空之主生下最高火，即众生之火，名为家主。

                第一火鸯耆罗是所有火的起源，是唯一之火。

            }//好多火啊。。。

            七位仙人之妻生室建陀：

            {

                阿修罗总是战胜天神。因陀罗想为军队找统帅。

                此时他听到妇女的叫声“来人救救我，让一个人做我的丈夫。”

                因陀罗发现有人抢夺这个女子，他救了她，并给她恩赐赐予她战无不胜的丈夫。

                此人也将是他军队的统帅。

                火在祭祀中看到仙人的妻子们，他爱上这些妻子。

                陀刹的女儿娑婆诃爱上了火。//陀刹真是女儿专业户。

                娑婆诃幻化为仙人六位妻子与火神交欢，将火神的精液洒在金盆里。

                这些精液聚在一起生出一个力量强大的孩子---室建陀（射出者）。

                七位仙人听说自己妻子与火神生子，便遗弃了六位妻子。

                娑婆诃一再解释。众友仙人也告知真相。然而仙人们依然遗弃他们的妻子。

                因陀罗为了避免室建陀超过自己，想杀死火神之子，却无力战胜，只能请求室建陀的保护。

                众神请求室建陀成为神中之王，因陀罗。然室建陀不想成为因陀罗，他选择成为因陀罗的臣仆。

                因陀罗将室建陀灌顶为天神军统帅。

                室建陀娶了之前因陀罗所救的少女。

                室建陀让被仙人抛弃妻子们拥有不朽的灵魂，受人崇拜，可以各种方式折磨16岁以下的孩子。

                室建陀还让自己亲生母亲娑婆诃和火神在一起。

                楼陀罗进入火中，乌玛进入娑婆诃，才生下室建陀，故而室建陀又被称为楼陀罗之子。//又变成了shiva和uma的儿子

                在天神和阿修罗的战斗中，室建陀杀死摩希舍，战胜敌人征服三界。

            }

        }

1. 黑天与坚战等人相遇。

2. 摩根德耶讲述的n篇插话：

    1) 业决定人死后的归宿。

    2) 善待婆罗门能让人复活的插话。

    3) 梵与王权的争论。

    4) 辩才仙女：

    {

        如何不偏离正法。

        古代火祭的戒规。

        牟尼达到最高解脱是什么？

    }

    5) 摩奴和鱼//印度的诺亚方舟

    6) 时代进程，路遇儿童毗湿奴，看到口中世界。

    7) 非法时代。

    8) 国王输给仙人。

    9) 只有乌龟记得也能升入天国。

    10) 尤腾伽帮助巨马之子杀死敦杜。

    11) 婆罗门经妇女指点见到猎人，让自己获救。

    12) 火的世系：鸯耆罗和五生生出世间无数种火。

    13) 七位仙人之妻，火神，娑婆诃，楼陀罗共同生室建陀。
