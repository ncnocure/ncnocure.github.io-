47-夺牛篇 札记 #摩诃婆罗多#

发表于 2015-07-26 更新于 2019-02-02 分类于 Text

#ifndef 47-夺牛篇.md

厉声：

{

    护民子：

    {

        空竹之死带来的连锁反应：

        {

            三穴王是空竹的手下败将，空竹死后，毗罗吒王独木难支。

            于是三穴王建议难敌攻打毗罗吒王，夺牛群。

            迦尔纳和难敌带领俱卢族出征帮助盟友三穴王掠夺牛群。

        }

        毗罗吒王攻打三穴国夺回牲畜，战争过程：

        {

            毗罗吒王安排坚战，怖军，无种，偕天上战场。//阿周那还在后宫教公主们跳舞。

            战争中，毗罗吒王被抓走。

            怖军营救毗罗吒王，俘虏三穴王。三穴国全军败退。

            战后毗罗吒赐给四子财宝，甚至打算将王位让给坚战。

        }

        同时，难敌与毗湿摩，德罗纳，继而那，慈悯，沙恭尼，难降等俱卢全套强人进犯毗罗吒王城，抢劫牛群，战争过程：

        {

            毗罗吒王之子胜地得知此事出战，巨苇为车夫。

            巨苇答应公主带回薄纱衣服。

            胜地遇敌之后吓得临阵脱逃，被巨苇强行拖回。

            巨苇让胜地做车夫，先取回弓箭，表明身份，'褪下手镯，束起卷发，吹响螺号，告知大地战神归来'。难敌方听出阿周那到来。

            '毗湿摩确认五子已经完成十三年又五个月零十二天的期限。'//这个章节的关键点，五子确定完成誓言，而难敌却没有归还王国。

            俱卢方分兵让难敌带着掠夺的牛群先回去。

            阿周那追着逃跑的难敌，德罗纳带人追在阿周那后面。

            阿周那打败难敌夺回牛群，陷入俱卢六将围攻中。

            阿周那开大了，AOE地图杀，而后单秒迦尔纳弟弟的胜战。

            迦尔纳开大，但被阿周那防反，只能败退回城。

            阿周那力战六强，帝释天带着三十三天诸神下凡围观。

            阿周那 v 慈悯，胜。

            阿周那 v 德罗纳，胜。

            阿周那 v 马嘶，胜。//马嘶断阿周那弓弦

            阿周那 v 迦尔纳，胜。//迦尔纳败退

            阿周那 v 4/100 百子，胜。

            阿周那开大，俱卢全线崩溃。

            阿周那 v 毗湿摩，胜。

            阿周那 v 难敌，胜。

            阿周那 v 象王，胜。//可以带入到魔戒里的精灵王子杀巨兽

            阿周那开嘲讽。

            难敌被嘲讽，停止逃跑再次攻击。//难敌也是条汉子

            其他六人回转。

            阿周那开大型控场技能，全地图恐惧。

            毗湿摩自带免疫恐惧，是唯一没被控的。

            王子趁机拿走俱卢方的衣服。//给公主们带的，这个b装的满分！

            阿周那离去，其他人赞他手下留情。

            阿周那归来让王子先进城，宣布王子的胜利。

        }

    }

}

 

#endif

##主要内容：

1. 空竹之死导致三穴城联合俱卢攻打毗罗吒。

2. 4/5五子出场帮助毗罗吒抢回牛群。

3. 六英战胜财。

##迦尔纳弟弟之死

阿周那对俱卢王室以及自己师傅还是手下留情的，而即非国王，又非皇室的迦尔纳弟弟就没这么好运。

这里的迦尔纳弟弟的身份并没有交代，可能是罗陀和胜车的儿子。
